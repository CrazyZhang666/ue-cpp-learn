# 14.2 实现技能

> **目标**：从零开始创建4个完整的GameplayAbility（近战攻击、远程投射物、闪现冲刺、范围治疗），并搭建技能输入绑定系统。

---

## 2.1 技能系统架构概览

在开始写代码之前，先理解GAS技能系统的工作流程：

```
玩家按下按键
    ↓
Enhanced Input Action 触发
    ↓
AbilityInputBindingComponent 将输入映射为 GameplayTag
    ↓
ASC (AbilitySystemComponent) 收到 GameplayTag
    ↓
查找匹配的 GameplayAbility（通过 AbilityTags 匹配）
    ↓
CanActivateAbility 检查：
  - 拥有该技能吗？        →❌ 没有 → 失败
  - 冷却中吗？            →❌ 冷却中 → 失败
  - 消耗足够吗？          →❌ 不够 → 失败
  - 被标签阻止吗？        →❌ 被Stun等阻止 → 失败
  - 所有条件通过          →✅
    ↓
ActivateAbility 执行技能逻辑
    ↓
技能结束（EndAbility）
```

## 2.2 技能输入绑定系统

### 2.2.1 核心问题

GAS的标准输入绑定方式是通过 `AbilitySystemComponent::AbilityLocalInputPressed`，需要手动在绑定代码中处理每个技能的ID。这在大型项目中极其繁琐。

**更好的方案**：使用GameplayTag作为中间层，让EnhancedInput的Action映射为GameplayTag，ASC根据AbilityTag自动激活对应技能。

### 2.2.2 创建输入绑定组件

创建文件 `Source/YourProject/Public/GAS/Input/RPGInputBindingComponent.h`：

```cpp
// ====================================================================
// 文件：Source/YourProject/Public/GAS/Input/RPGInputBindingComponent.h
// 描述：增强输入绑定组件 — 将Enhanced Input与GAS的GameplayTag系统桥接
// ====================================================================

#pragma once

#include "CoreMinimal.h"
#include "Components/ActorComponent.h"
#include "EnhancedInputComponent.h"                 // 增强输入组件
#include "GameplayTagContainer.h"                    // GameplayTag容器
#include "RPGInputBindingComponent.generated.h"

// 前向声明
class UInputAction;     // 增强输入资产
class UInputMappingContext;  // 输入映射上下文
class UAbilitySystemComponent;

/**
 * URPGInputBindingComponent — 技能输入绑定组件
 *
 * 工作流程：
 * 1. 在编辑器中配置 InputAction → GameplayTag 的映射
 * 2. 注册到EnhancedInputComponent
 * 3. 玩家按下按键 → InputAction触发 → 查找对应GameplayTag → 通知ASC激活技能
 */
UCLASS(ClassGroup = (GAS), meta = (BlueprintSpawnableComponent))
class YOURPROJECT_API URPGInputBindingComponent : public UActorComponent
{
    GENERATED_BODY()

public:
    URPGInputBindingComponent();

    // ================================================================
    // 配置：InputAction 到 GameplayTag 的映射条目
    // ================================================================

    /** 单个输入映射：一个InputAction对应一个GameplayTag */
    USTRUCT(BlueprintType)
    struct FInputActionTagBinding
    {
        GENERATED_BODY()

        /** 增强输入资产（如 IA_Attack、IA_Jump） */
        UPROPERTY(EditDefaultsOnly, BlueprintReadOnly)
        TObjectPtr<const UInputAction> InputAction;

        /** 按下时发送的GameplayTag（如 Input.Attack.Light） */
        UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, meta = (Categories = "InputTag"))
        FGameplayTag InputTag_Pressed;

        /** 松开时发送的GameplayTag（如 Input.Attack.Light.Released） */
        UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, meta = (Categories = "InputTag"))
        FGameplayTag InputTag_Released;

        /**
         * 按住时触发的GameplayTag（可选，用于蓄力技能）
         * 如果为空，则不绑定Held事件
         */
        UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, meta = (Categories = "InputTag"))
        FGameplayTag InputTag_Held;
    };

    // ================================================================
    // 公共接口
    // ================================================================

    /**
     * 初始化输入绑定
     * 必须在SetupPlayerInputComponent中调用
     * @param InASC         — 角色的AbilitySystemComponent
     * @param InputComponent — 角色的EnhancedInputComponent
     */
    void InitializeInputBindings(
        UAbilitySystemComponent* InASC,
        UEnhancedInputComponent* InputComponent);

    /**
     * 获取ASC引用（用于外部访问）
     */
    UAbilitySystemComponent* GetAbilitySystemComponent() const { return ASC; }

    /**
     * 手动触发技能输入（用于AI或脚本驱动）
     * @param InputTag — 要触发的InputTag（如 Input.Attack.Light）
     */
    UFUNCTION(BlueprintCallable, Category = "Input")
    void InjectInputPressed(const FGameplayTag& InputTag);

    /**
     * 手动松开技能输入
     */
    UFUNCTION(BlueprintCallable, Category = "Input")
    void InjectInputReleased(const FGameplayTag& InputTag);

protected:
    // ================================================================
    // 编辑器配置
    // ================================================================

    /** 输入映射上下文（包含所有IA → Tag的映射） */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Input|Config")
    TArray<FInputActionTagBinding> InputBindings;

    /**
     * 是否启用调试日志
     * 开启后每次按键都会打印日志
     * ⚠️ 打包发布时记得关闭（大量日志影响性能）
     */
    UPROPERTY(EditDefaultsOnly, Category = "Input|Debug")
    bool bShowInputDebugLog = false;

private:
    // ================================================================
    // 内部状态
    // ================================================================

    /** AbilitySystemComponent引用 */
    UPROPERTY()
    TObjectPtr<UAbilitySystemComponent> ASC;

    /**
     * 处理InputAction按下事件
     * 查找InputAction对应的GameplayTag，通知ASC
     */
    void OnInputPressed(const FInputActionInstance& Instance);

    /**
     * 处理InputAction松开事件
     */
    void OnInputReleased(const FInputActionInstance& Instance);

    /**
     * 处理InputAction按住事件（每帧触发）
     */
    void OnInputHeld(const FInputActionInstance& Instance);

    /** 找到输入标签对应的技能并激活 */
    void TryActivateAbilityByTag(const FGameplayTag& InputTag, bool bPressed);

    /** 从InputAction查找对应的绑定条目 */
    const FInputActionTagBinding* FindBinding(const UInputAction* Action) const;
};
```

### 2.2.3 创建输入绑定组件实现

创建文件 `Source/YourProject/Private/GAS/Input/RPGInputBindingComponent.cpp`：

```cpp
// ====================================================================
// 文件：Source/YourProject/Private/GAS/Input/RPGInputBindingComponent.cpp
// 描述：增强输入绑定组件实现
// ====================================================================

#include "GAS/Input/RPGInputBindingComponent.h"
#include "AbilitySystemComponent.h"
#include "EnhancedInputComponent.h"
#include "EnhancedInputSubsystems.h"
#include "InputAction.h"

URPGInputBindingComponent::URPGInputBindingComponent()
{
    PrimaryComponentTick.bCanEverTick = false;  // 不需要帧更新（事件驱动）
}

void URPGInputBindingComponent::InitializeInputBindings(
    UAbilitySystemComponent* InASC,
    UEnhancedInputComponent* InputComponent)
{
    // ================================================================
    // 第一步：保存ASC引用
    // ================================================================
    ASC = InASC;

    if (!IsValid(ASC) || !IsValid(InputComponent))
    {
        UE_LOG(LogTemp, Error,
            TEXT("RPGInputBindingComponent: Invalid ASC or InputComponent"));
        return;
    }

    // ================================================================
    // 第二步：绑定每个InputAction到对应的回调
    // ================================================================

    for (const FInputActionTagBinding& Binding : InputBindings)
    {
        // 检查InputAction是否有效
        if (!IsValid(Binding.InputAction))
        {
            UE_LOG(LogTemp, Warning,
                TEXT("RPGInputBindingComponent: Skipping binding with null InputAction"));
            continue;
        }

        // --- 绑定按下事件 ---
        if (Binding.InputTag_Pressed.IsValid())
        {
            InputComponent->BindAction(
                Binding.InputAction,       // 要绑定的增强输入资产
                ETriggerEvent::Started,     // 触发时机：按键刚按下
                this,                       // 回调对象
                &URPGInputBindingComponent::OnInputPressed  // 回调函数
            );
        }

        // --- 绑定松开事件 ---
        if (Binding.InputTag_Released.IsValid())
        {
            InputComponent->BindAction(
                Binding.InputAction,
                ETriggerEvent::Completed,   // 触发时机：按键松开
                this,
                &URPGInputBindingComponent::OnInputReleased
            );
        }

        // --- 绑定按住事件（可选，每帧触发）---
        if (Binding.InputTag_Held.IsValid())
        {
            InputComponent->BindAction(
                Binding.InputAction,
                ETriggerEvent::Triggered,   // 触发时机：按住时每帧
                this,
                &URPGInputBindingComponent::OnInputHeld
            );
        }
    }

    UE_LOG(LogTemp, Log,
        TEXT("RPGInputBindingComponent: Bound %d input action(s)"), InputBindings.Num());
}

void URPGInputBindingComponent::OnInputPressed(const FInputActionInstance& Instance)
{
    // 查找这个InputAction对应的绑定配置
    const UInputAction* SourceAction = Instance.GetSourceAction();
    const FInputActionTagBinding* Binding = FindBinding(SourceAction);

    if (Binding)
    {
        if (bShowInputDebugLog)
        {
            UE_LOG(LogTemp, Log, TEXT("Input PRESSED: %s → Tag: %s"),
                *SourceAction->GetName(), *Binding->InputTag_Pressed.ToString());
        }

        // ✅ 正确做法：通过ASC的AbilityLocalInputPressed间接激活技能
        // ❌ 错误做法：直接查找GA并调用ActivateAbility（绕过了GAS的权限检查）
        TryActivateAbilityByTag(Binding->InputTag_Pressed, true);
    }
}

void URPGInputBindingComponent::OnInputReleased(const FInputActionInstance& Instance)
{
    const UInputAction* SourceAction = Instance.GetSourceAction();
    const FInputActionTagBinding* Binding = FindBinding(SourceAction);

    if (Binding)
    {
        if (bShowInputDebugLog)
        {
            UE_LOG(LogTemp, Log, TEXT("Input RELEASED: %s → Tag: %s"),
                *SourceAction->GetName(), *Binding->InputTag_Released.ToString());
        }

        TryActivateAbilityByTag(Binding->InputTag_Released, false);
    }
}

void URPGInputBindingComponent::OnInputHeld(const FInputActionInstance& Instance)
{
    const UInputAction* SourceAction = Instance.GetSourceAction();
    const FInputActionTagBinding* Binding = FindBinding(SourceAction);

    if (Binding && Binding->InputTag_Held.IsValid())
    {
        // Held事件每帧触发，通常用于蓄力技能
        // 这里简单地重复发送按下事件
        TryActivateAbilityByTag(Binding->InputTag_Held, true);
    }
}

void URPGInputBindingComponent::TryActivateAbilityByTag(const FGameplayTag& InputTag, bool bPressed)
{
    // ================================================================
    // 方法一：使用AbilityInputTagPressed（推荐用于简单项目）
    // ================================================================
    // ASC维护了一个按InputTag分类的技能索引
    // 调用PressInputID会触发所有匹配该Tag的GA的InputPressed回调

    if (!IsValid(ASC))
    {
        return;
    }

    // 获取所有被授予且拥有指定InputTag的技能
    TArray<FGameplayAbilitySpecHandle> AbilitiesToActivate;
    ASC->FindAllAbilitiesWithInputID(ASC->GetInputID(InputTag), AbilitiesToActivate);

    // 激活找到的技能
    for (const FGameplayAbilitySpecHandle& Handle : AbilitiesToActivate)
    {
        if (Handle.IsValid())
        {
            // 让ASC尝试激活此技能
            // TryActivateAbility 会检查所有条件（标签、冷却、消耗等）
            if (!ASC->TryActivateAbility(Handle))
            {
                // 激活失败（可能因为冷却中、消耗不足、被阻止等）
                // 这里可以播放失败音效
                if (bShowInputDebugLog)
                {
                    UE_LOG(LogTemp, Warning,
                        TEXT("RPGInputBindingComponent: Failed to activate ability with tag %s"),
                        *InputTag.ToString());
                }
            }
        }
    }
}

const URPGInputBindingComponent::FInputActionTagBinding*
URPGInputBindingComponent::FindBinding(const UInputAction* Action) const
{
    // 遍历所有绑定，查找匹配的InputAction
    for (const FInputActionTagBinding& Binding : InputBindings)
    {
        if (Binding.InputAction == Action)
        {
            return &Binding;
        }
    }

    // 没有找到匹配的绑定
    return nullptr;
}

void URPGInputBindingComponent::InjectInputPressed(const FGameplayTag& InputTag)
{
    // AI/脚本驱动接口：手动触发技能输入
    TryActivateAbilityByTag(InputTag, true);
}

void URPGInputBindingComponent::InjectInputReleased(const FGameplayTag& InputTag)
{
    // AI/脚本驱动接口：手动触发技能松开
    TryActivateAbilityByTag(InputTag, false);
}
```

```cpp
void ARPGCharacter::SetupPlayerInputComponent(UInputComponent* PlayerInputComponent)
{
    Super::SetupPlayerInputComponent(PlayerInputComponent);

    // 将UInputComponent转换为EnhancedInputComponent
    UEnhancedInputComponent* EnhancedInput = Cast<UEnhancedInputComponent>(PlayerInputComponent);

    if (IsValid(EnhancedInput) && IsValid(InputBindingComponent))
    {
        // 初始化输入绑定
        InputBindingComponent->InitializeInputBindings(AbilitySystemComponent, EnhancedInput);
    }
    else
    {
        UE_LOG(LogTemp, Error,
            TEXT("RPGCharacter: Failed to setup input — EnhancedInputComponent or InputBindingComponent is null"));
    }
}
```

```cpp
// ====================================================================
// 文件：Source/YourProject/Public/GAS/Ability/RPGGameplayAbility.h
// 描述：RPG技能基类 — 所有自定义GameplayAbility的父类
// ====================================================================

#pragma once

#include "CoreMinimal.h"
#include "Abilities/GameplayAbility.h"             // GA基类
#include "GameplayTagContainer.h"                   // GameplayTag容器
#include "RPGGameplayAbility.generated.h"

// 前向声明
class UGameplayEffect;          // GE基类
class UAnimMontage;             // 动画蒙太奇

/**
 * URPGGameplayAbility — RPG技能基类
 *
 * 为所有技能提供统一的配置项和辅助函数：
 * 1. 技能输入标签（用于输入绑定匹配）
 * 2. 冷却GE和消耗GE的配置
 * 3. 技能动画蒙太奇
 * 4. 激活条件检查（死亡、状态等）
 */
UCLASS()
class YOURPROJECT_API URPGGameplayAbility : public UGameplayAbility
{
    GENERATED_BODY()

public:
    URPGGameplayAbility();

    // ================================================================
    // 编辑器配置属性
    // ================================================================

    /**
     * 技能的输入标签
     * 用于在输入绑定时匹配。例如：
     * - Input.Attack.Light → 轻攻击
     * - Input.Skill.Fireball → 火球术
     * - Input.Skill.Dash → 闪现
     */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Input")
    FGameplayTag InputTag;

    /**
     * 技能的类别标签
     * 用于区分技能类型，例如：
     * - Ability.Type.Attack → 攻击类技能
     * - Ability.Type.Heal → 治疗类技能
     * - Ability.Type.Movement → 移动类技能
     */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Tags")
    FGameplayTagContainer AbilityTypeTags;

    /**
     * 激活此技能时，自动添加到角色的GameplayTag
     * 用于阻止其他技能激活，例如：
     * - 攻击时添加 Ability.State.Attacking，阻止新的攻击
     * - 施法时添加 Ability.State.Casting，阻止移动类技能
     */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Tags")
    FGameplayTagContainer ActivationOwnedTags;

    /**
     * 阻止激活的GameplayTag
     * 如果角色拥有以下任何Tag，技能无法激活：
     * - State.Dead → 死亡时阻止所有技能
     * - State.Stunned → 眩晕时阻止所有技能
     * - State.Silenced → 沉默时阻止施法技能
     */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Tags")
    FGameplayTagContainer ActivationBlockedTags;

    /**
     * 激活时自动应用到角色的GameplayTag（包括技能持续时间）
     */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Tags")
    FGameplayTagContainer ActivateOwnedTags;

    // ================================================================
    // 冷却相关配置
    // ================================================================

    /** 冷却GE类（Cooldown GameplayEffect） */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Cooldown")
    TSubclassOf<UGameplayEffect> CooldownEffectClass;

    /** 冷却时间（秒）— 仅在CooldownEffectClass未设置时生效 */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Cooldown")
    FScalableFloat CooldownDuration;

    /** 冷却标签（如 Ability.Cooldown.Fireball） */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Cooldown")
    FGameplayTagContainer CooldownTags;

    // ================================================================
    // 消耗相关配置
    // ================================================================

    /** 消耗GE类（Cost GameplayEffect） */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Cost")
    TSubclassOf<UGameplayEffect> CostEffectClass;

    /** 法力消耗量（仅在CostEffectClass未设置时生效） */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Cost")
    FScalableFloat ManaCost;

    /** 体力消耗量 */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Cost")
    FScalableFloat StaminaCost;

    // ================================================================
    // 动画相关配置
    // ================================================================

    /** 技能播放的动画蒙太奇 */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Animation")
    TObjectPtr<UAnimMontage> AbilityMontage;

    /** 动画播放速率 */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Animation")
    float MontagePlayRate = 1.0f;

    /** 动画起始Section名称（用于从特定帧开始播放） */
    UPROPERTY(EditDefaultsOnly, BlueprintReadOnly, Category = "Ability|Animation")
    FName MontageStartSection;

    // ================================================================
    // 覆写函数
    // ================================================================

    /**
     * 检查技能是否可以激活
     * 在GAS内部调用，在ActivateAbility之前
     * @return true = 技能可以激活
     */
    virtual bool CanActivateAbility(
        const FGameplayAbilitySpecHandle Handle,
        const FGameplayAbilityActorInfo* ActorInfo,
        const FGameplayTagContainer* SourceTags,
        const FGameplayTagContainer* TargetTags,
        FGameplayTagContainer* OptionalRelevantTags) const override;

    /**
     * 激活技能时的主入口
     * 子类覆写此函数实现具体技能逻辑
     */
    virtual void ActivateAbility(
        const FGameplayAbilitySpecHandle Handle,
        const FGameplayAbilityActorInfo* ActorInfo,
        const FGameplayAbilityActivationInfo ActivationInfo,
        const FGameplayEventData* TriggerEventData) override;

    /**
     * 结束技能
     * 执行清理工作（移除标签、结束Task、销毁Actor等）
     */
    virtual void EndAbility(
        const FGameplayAbilitySpecHandle Handle,
        const FGameplayAbilityActorInfo* ActorInfo,
        const FGameplayAbilityActivationInfo ActivationInfo,
        bool bReplicateEndAbility,
        bool bWasCancelled) override;

    // ================================================================
    // 辅助函数
    // ================================================================

    /**
     * 应用冷却效果（在ConfirmAbility或CommitAbility后调用）
     */
    UFUNCTION(BlueprintCallable, Category = "Ability")
    void ApplyCooldown(const FGameplayAbilitySpecHandle Handle,
        const FGameplayAbilityActorInfo* ActorInfo,
        const FGameplayAbilityActivationInfo ActivationInfo);

    /**
     * 应用消耗效果
     */
    UFUNCTION(BlueprintCallable, Category = "Ability")
    void ApplyCost(const FGameplayAbilitySpecHandle Handle,
        const FGameplayAbilityActorInfo* ActorInfo,
        const FGameplayAbilityActivationInfo ActivationInfo);

    /**
     * 获取源角色的AbilitySystemComponent
     */
    UFUNCTION(BlueprintCallable, Category = "Ability")
    UAbilitySystemComponent* GetSourceASC() const;

    /**
     * 获取源角色（Avatar Actor）
     */
    UFUNCTION(BlueprintCallable, Category = "Ability")
    AActor* GetSourceActor() const;

    /**
     * 获取源角色的AttributeSet
     */
    UFUNCTION(BlueprintCallable, Category = "Ability")
    class URPGAttributeSet* GetSourceAttributeSet() const;
};
```

### 2.3.2 创建GA基类实现

创建文件 `Source/YourProject/Private/GAS/Ability/RPGGameplayAbility.cpp`：

```cpp
// ====================================================================
// 文件：Source/YourProject/Private/GAS/Ability/RPGGameplayAbility.cpp
// 描述：RPG技能基类实现
// ====================================================================

#include "GAS/Ability/RPGGameplayAbility.h"
#include "AbilitySystemComponent.h"
#include "AbilitySystemGlobals.h"
#include "GAS/AttributeSet/RPGAttributeSet.h"
#include "GameFramework/Character.h"

URPGGameplayAbility::URPGGameplayAbility()
{
    // ================================================================
    // 默认配置
    // ================================================================

    // 默认实例化策略：每次激活创建新实例
    // InstancedPerActor：每个角色只有一个实例，但每次激活会重新初始化
    // InstancedPerExecution：每次激活都创建全新的实例（用于需要完全隔离的技能）
    // NonInstanced：不创建实例（用于简单的被动技能）
    InstancingPolicy = EGameplayAbilityInstancingPolicy::InstancedPerActor;

    // 网络执行策略
    // LocalPredicted：客户端预测执行（减少延迟感，适合动作游戏）
    // LocalOnly：仅在本地执行（纯视觉效果技能）
    // ServerInitiated：服务器触发
    // ServerOnly：仅在服务器执行
    NetExecutionPolicy = EGameplayAbilityNetExecutionPolicy::LocalPredicted;

    // 动画播放速率
    MontagePlayRate = 1.0f;
}

    const FGameplayTagContainer* SourceTags,
    const FGameplayTagContainer* TargetTags,
    FGameplayTagContainer* OptionalRelevantTags) const
{
    // ================================================================
    // 首先调用父类的检查（父类检查基础条件）
    // ================================================================
    if (!Super::CanActivateAbility(Handle, ActorInfo, SourceTags, TargetTags, OptionalRelevantTags))
    {
        return false;
    }

    // ================================================================
    // 自定义检查：是否被BlockedTags阻止？
    // ================================================================
    // 从ActorInfo中获取ASC，检查角色当前是否拥有阻止标签

    if (ActorInfo && ActorInfo->AbilitySystemComponent.IsValid())
    {
        const UAbilitySystemComponent* ASC = ActorInfo->AbilitySystemComponent.Get();

        // HasAnyMatchingGameplayTags：检查ASC是否拥有任意一个BlockedTag
        if (ASC->HasAnyMatchingGameplayTags(ActivationBlockedTags))
        {
            // 角色处于被阻止状态（如死亡、眩晕等）
            // 注意：如果正在冷却中，GAS父类会检查，不需要在这里处理
            if (OptionalRelevantTags)

                OptionalRelevantTags->AppendTags(ActivationBlockedTags);
            }
            return false;  // ❌ 技能被阻止
        }
    }

    // ✅ 所有检查通过
    return true;
}
```

```cpp
// ====================================================================
// 文件：Source/YourProject/Private/GAS/Ability/GA_MeleeAttack.cpp
// 描述：近战攻击GameplayAbility实现
// ====================================================================

#include "GAS/Ability/GA_MeleeAttack.h"
#include "Abilities/Tasks/AbilityTask_PlayMontageAndWait.h"    // 动画Task
#include "Abilities/Tasks/AbilityTask_WaitGameplayEvent.h"     // 事件Task
#include "AbilitySystemComponent.h"
#include "AbilitySystemGlobals.h"
#include "GAS/AttributeSet/RPGAttributeSet.h"
#include "Engine/OverlapResult.h"                              // 重叠检测结果
#include "Kismet/KismetSystemLibrary.h"                        // 球形检测等工具
#include "GameFramework/Character.h"

UGA_MeleeAttack::UGA_MeleeAttack()
{

    // InputTag 在蓝图子类中设置（如 Input.Attack.Light）
    // AbilityTypeTags 标识这是一个攻击技能
    AbilityTypeTags.AddTag(FGameplayTag::RequestGameplayTag("Ability.Type.Attack"));
    AbilityTypeTags.AddTag(FGameplayTag::RequestGameplayTag("Ability.Type.Melee"));

    // 激活时给角色添加此标签（阻止新的近战攻击同时执行）
    ActivationOwnedTags.AddTag(FGameplayTag::RequestGameplayTag("Ability.State.Attacking"));

    // 被以下标签阻止时无法激活
    ActivationBlockedTags.AddTag(FGameplayTag::RequestGameplayTag("State.Dead"));
    ActivationBlockedTags.AddTag(FGameplayTag::RequestGameplayTag("State.Stunned"));

    // --- 默认参数 ---
    AttackAngle = 90.0f;        // 90度扇形
    AttackRadius = 200.0f;      // 2米攻击范围
    SkillMultiplier = 1.0f;     // 100%攻击力
    KnockbackDistance = 100.0f; // 100厘米击退

    // 伤害事件标签（对应动画蓝图中的GameplayTag通知）
    DamageEventTag = FGameplayTag::RequestGameplayTag("Event.Melee.DamageCheck");

    // 网络策略：服务器负责伤害检测，客户端负责播放动画和特效
    // ⚠️ 对于近战攻击，建议使用 ServerOnly 或 LocalPredicted
    // ServerOnly = 仅在服务器检测伤害（安全但延迟）
    // LocalPredicted = 客户端预测动画，服务器验证伤害（流畅且安全）

}

void UGA_MeleeAttack::ActivateAbility(
    const FGameplayAbilitySpecHandle Handle,
    const FGameplayAbilityActorInfo* ActorInfo,
    const FGameplayAbilityActivationInfo ActivationInfo,
    const FGameplayEventData* TriggerEventData)
{
    // ================================================================
    // 第一步：调用父类（基类处理通用逻辑）
    // ================================================================
    Super::ActivateAbility(Handle, ActorInfo, ActivationInfo, TriggerEventData);

    // ================================================================
    // 第二步：检查是否有动画蒙太奇
    // ================================================================
    if (!AbilityMontage)
    {
        UE_LOG(LogTemp, Error, TEXT("GA_MeleeAttack: No AbilityMontage set! Ending ability."));
        EndAbility(Handle, ActorInfo, ActivationInfo, true, true);
        return;
    }

    // ================================================================
    // 第三步：提交技能（检查冷却和消耗）
    // ================================================================
    // CommitAbility 是GAS内置函数：
    // 1. 检查Cooldown标签（是否在冷却中）
    // 2. 检查Cost（法力、体力等是否足够）
    // 3. 如果检查通过 → 应用冷却GE和消耗GE
    // 4. 如果检查失败 → 不会应用任何效果，技能无法继续
    // ================================================================
    if (!CommitAbility(Handle, ActorInfo, ActivationInfo))
    {
        // ❌ 提交失败：冷却中或消耗不足
        UE_LOG(LogTemp, Warning, TEXT("GA_MeleeAttack: CommitAbility failed (cooldown or cost)"));
        EndAbility(Handle, ActorInfo, ActivationInfo, true, true);
        return;
    }

    // ================================================================
    // 第四步：播放攻击动画（使用AbilityTask）
    // ================================================================
    // AbilityTask_PlayMontageAndWait：
    // - 异步播放动画蒙太奇
    // - OnCompleted：动画正常播放完成
    // - OnBlendOut：动画淡出
    // - OnInterrupted：动画被打断（例如受到控制效果）
    // - OnCancelled：技能被取消
    // ⚠️ AbilityTask在EndAbility时自动清理（如果调用了EndAbilityTask）

    MontageTask = UAbilityTask_PlayMontageAndWait::CreatePlayMontageAndWaitProxy(
        this,                           // OwningAbility（拥有这个Task的GA）
        NAME_None,                      // Task名称（空=默认）
        AbilityMontage,                 // 动画蒙太奇
        MontagePlayRate,                // 播放速率
        MontageStartSection,            // 起始Section
        false,                          // 是否在技能结束时停止动画（false=自然播放完毕）
        1.0f                            // RootMotion缩放
    );

    // --- 绑定动画完成回调 ---
    MontageTask->OnCompleted.AddDynamic(this, &UGA_MeleeAttack::OnMontageCompleted);
    MontageTask->OnInterrupted.AddDynamic(this, &UGA_MeleeAttack::OnMontageInterrupted);

    // --- 激活Task（开始异步执行） ---
    MontageTask->ReadyForActivation();

    // ================================================================
    // 第五步：等待伤害检测事件（动画通知触发）
    // ================================================================
    // 在动画蓝图中，在武器挥出的关键帧添加 AnimNotify：
    // 在Notify中通过蓝图发送GameplayEvent，Tag为 Event.Melee.DamageCheck
    //
    // 为什么不用 AnimNotify 直接调用C++函数？
    // 原因：AnimNotify 在客户端和服务器都执行
    // 但我们只想在服务器端做伤害检测（client can cheat）
    // AbilityTask_WaitGameplayEvent 能正确处理网络复制

    WaitDamageEventTask = UAbilityTask_WaitGameplayEvent::WaitGameplayEvent(
        this,                           // OwningAbility
        DamageEventTag,                 // 等待这个GameplayTag
        nullptr,                        // 可选的目标Actor（null=任意来源）
        false,                          // 是否只触发一次（false=每次都触发）
        false                           // 是否只匹配精确Tag（false=也匹配父标签）
    );

    // --- 绑定事件接收回调 ---
    WaitDamageEventTask->EventReceived.AddDynamic(this, &UGA_MeleeAttack::OnDamageEventReceived);

    // --- 激活Task ---
    WaitDamageEventTask->ReadyForActivation();

    // ================================================================
    // 注意：技能现在处于"激活但等待"状态
    // - 动画正在播放
    // - 等待动画通知事件
    // - EndAbility 会在动画完成或被打断时调用
    // ================================================================
}

    // ================================================================

    ActorsToIgnore.Add(SourceActor);  // 忽略自身

    TArray<FOverlapResult> OverlapResults;

    // 使用球形重叠检测
    bool bHasOverlap = UKismetSystemLibrary::SphereOverlapActors(
        SourceActor->GetWorld(),
        SourceLocation,
        AttackRadius,                   // 检测半径
        ObjectTypes,                    // 检测对象类型
        APawn::StaticClass(),           // 指定类过滤器
        ActorsToIgnore,                 // 忽略列表
        OverlapResults                  // [输出] 重叠结果
    );

    if (!bHasOverlap)
    {
        return Result;  // 范围内没有目标
    }

    // ================================================================
    // 第二步：角度过滤 — 只保留在前方扇形区域内的目标
    // ================================================================

    const float HalfAngle = AttackAngle * 0.5f;        // 半角（cos比较需要半角）
    const float CosHalfAngle = FMath::Cos(FMath::DegreesToRadians(HalfAngle));

    for (const FOverlapResult& Overlap : OverlapResults)
    {
        AActor* HitActor = Overlap.GetActor();
        if (!IsValid(HitActor))
        {
            continue;
        }

        // --- 计算目标相对于攻击者的方向 ---
        const FVector DirectionToTarget = (HitActor->GetActorLocation() - SourceLocation).GetSafeNormal();
        const float DotProduct = FVector::DotProduct(ForwardVector, DirectionToTarget);

        // --- 点积比较：cos(角度)越大，角度越小 ---
        // dot > CosHalfAngle 意味着目标在扇形角度内
        if (DotProduct > CosHalfAngle)
        {
            // --- 额外过滤：排除友方角色 ---
            // 在实际项目中，你需要根据Team ID、Faction等判断
            // 这里简化：所有检测到的非自身Pawn都是敌人
            Result.Add(HitActor);
        }
    }

    return Result;
}

void UGA_MeleeAttack::ApplyDamageToTarget(AActor* Target)
{
    // ================================================================
    // 第一步：获取目标的ASC
    // ================================================================
    // UAbilitySystemGlobals::GetAbilitySystemComponentFromActor
    // 是一个便捷函数，从任意Actor上查找ASC（通过IAbilitySystemInterface接口）

    UAbilitySystemComponent* TargetASC =
        UAbilitySystemGlobals::GetAbilitySystemComponentFromActor(Target);

    if (!IsValid(TargetASC))
    {
        UE_LOG(LogTemp, Warning, TEXT("GA_MeleeAttack: Target %s has no ASC"),
            *Target->GetName());
        return;  // 目标没有GAS（可能是环境物体、装饰物等）
    }

    // ================================================================
    // 第二步：获取源ASC并创建GE Spec
    // ================================================================
    UAbilitySystemComponent* SourceASC = GetSourceASC();
    if (!IsValid(SourceASC))
    {
        return;
    }

    // --- 创建GE上下文（包含谁攻击了谁的信息） ---
    FGameplayEffectContextHandle DamageContext = SourceASC->MakeEffectContext();
    DamageContext.AddSourceObject(this);  // 标记技能为来源

    // --- 基于DamageEffectClass创建GE Spec ---
    FGameplayEffectSpecHandle DamageSpecHandle = SourceASC->MakeOutgoingSpec(
        DamageEffectClass,     // 伤害GE类
        GetAbilityLevel(),     // 技能等级（影响GE的ScalableFloat）
        DamageContext          // 上下文
    );

    if (!DamageSpecHandle.IsValid())
    {
        UE_LOG(LogTemp, Error, TEXT("GA_MeleeAttack: Failed to create damage spec"));
        return;
    }

    // ================================================================
    // 第三步：设置SetByCaller传递技能倍率
    // ================================================================
    // 这个值会被DamageCalculation MMC读取
    // 公式：BaseDamage = AttackPower × SkillMultiplier

    FGameplayEffectSpec* DamageSpec = DamageSpecHandle.Data.Get();
    const FGameplayTag MultiplierTag = FGameplayTag::RequestGameplayTag(
        "Data.SkillMultiplier");
    DamageSpec->SetSetByCallerMagnitude(MultiplierTag, SkillMultiplier);

    // ================================================================
    // 第四步：设置攻击类型标签（用于防御计算、抗性匹配等）
    // ================================================================
    // 例如：标记为物理伤害，这样防御方的物理抗性生效
    DamageSpec->DynamicGrantedTags.AddTag(
        FGameplayTag::RequestGameplayTag("Combat.Damage.Physical"));

    // 如果是元素伤害，则添加对应的标签：
    // DamageSpec->DynamicGrantedTags.AddTag(
    //     FGameplayTag::RequestGameplayTag("Combat.Damage.Fire"));

    // ================================================================
    // 第五步：应用GE到目标
    // ================================================================
    // ApplyGameplayEffectSpecToTarget 将伤害GE应用到目标角色
    // GE内部的MMC会在此刻计算最终伤害值

    FActiveGameplayEffectHandle ActiveGEHandle =
        SourceASC->ApplyGameplayEffectSpecToTarget(*DamageSpec, TargetASC);

    if (ActiveGEHandle.WasSuccessfullyApplied())
    {
        UE_LOG(LogTemp, Log, TEXT("GA_MeleeAttack: Damage applied to %s"), *Target->GetName());

        // ================================================================
        // 第六步：击退效果（可选）
        // ================================================================
        if (KnockbackDistance > 0.0f)
        {
            // 计算击退方向（从攻击者指向目标）
            FVector KnockbackDirection = (Target->GetActorLocation() -
                GetSourceActor()->GetActorLocation()).GetSafeNormal();
            KnockbackDirection.Z = 0.0f;  // 只在水平面击退

            // 移动目标
            Target->SetActorLocation(
                Target->GetActorLocation() + KnockbackDirection * KnockbackDistance,
                true   // bSweep：检查碰撞（防止推入墙壁）
            );
        }
    }
}
```

```cpp
#include "GAS/Projectile/RPGProjectile.h"

    // 默认配置
    // ================================================================
    bDestroyOnHit = true;
    bDestroyOnMaxRange = true;
    MaxLifetime = 10.0f;
    MaxRange = 5000.0f;  // 50米最大射程

    // 网络：投射物通常只在服务器上有权威性
    // 客户端通过复制看到投射物位置，但伤害由服务器处理
    SetReplicates(true);
    SetReplicateMovement(true);

void ARPGProjectile::BeginPlay()
{
    Super::BeginPlay();

    // 绑定碰撞事件
    CollisionSphere->OnComponentBeginOverlap.AddDynamic(
        this, &ARPGProjectile::OnSphereOverlap);

    // 设置自毁定时器
    if (MaxLifetime > 0.0f)
    {
        SetLifeSpan(MaxLifetime);
    }

    // 记录发射位置（用于距离检查）
    SpawnLocation = GetActorLocation();
}

void ARPGProjectile::InitializeProjectile(
    UAbilitySystemComponent* InSourceASC,
    TSubclassOf<UGameplayEffect> InDamageEffectClass,
    float InSpeed,
    float InMaxRange,
    float InSkillMultiplier)
{
    // 保存攻击者ASC（弱引用）
    SourceASC = InSourceASC;

    // 保存伤害GE类
    DamageEffectClass = InDamageEffectClass;

    // 设置飞行速度
    if (ProjectileMovement)
    {
        ProjectileMovement->InitialSpeed = InSpeed;
        ProjectileMovement->MaxSpeed = InSpeed;
        ProjectileMovement->Velocity = GetActorForwardVector() * InSpeed;
    }

    // 设置最大射程
    MaxRange = InMaxRange;

    // 设置技能倍率
    SkillMultiplier = InSkillMultiplier;

    // 将发射者加入忽略列表（防止打到自己）
    if (AActor* SourceActor = InSourceASC->GetOwnerActor())
    {
        IgnoredActors.Add(SourceActor);
        CollisionSphere->IgnoreActorWhenMoving(SourceActor, true);
    }
}

void ARPGProjectile::Tick(float DeltaTime)
{
    Super::Tick(DeltaTime);

    // 检查是否超过最大射程
    if (bDestroyOnMaxRange)
    {
        const float DistanceTraveled = FVector::Dist(GetActorLocation(), SpawnLocation);
        if (DistanceTraveled >= MaxRange)
        {
            Destroy();  // 超出射程，销毁
        }
    }
}

void ARPGProjectile::OnSphereOverlap(
    UPrimitiveComponent* OverlappedComponent,
    AActor* OtherActor,
    UPrimitiveComponent* OtherComp,
    int32 OtherBodyIndex,
    bool bFromSweep,
    const FHitResult& SweepResult)
{
    // ================================================================
    // 过滤：忽略无效目标
    // ================================================================

    // 无效Actor（已经被销毁的）
    if (!IsValid(OtherActor))
    {
        return;
    }

    // 忽略列表中的Actor（包括发射者自己）
    if (IgnoredActors.Contains(OtherActor))
    {
        return;
    }

    // 已经命中过的Actor（防止穿透模式重复命中）
    if (HitActors.Contains(OtherActor))
    {
        return;
    }

    // ================================================================
    // 仅在服务器端处理伤害
    // ================================================================
    if (GetLocalRole() == ROLE_Authority)
    {
        ApplyDamageToTarget(OtherActor);
    }

    // 记录命中
    HitActors.Add(OtherActor);

    // 播放命中特效（所有端都播放）
    PlayHitEffect(SweepResult.Location, SweepResult.ImpactNormal.Rotation());

    // 碰撞后是否销毁
    if (bDestroyOnHit)
    {
        Destroy();
    }
}

void ARPGProjectile::ApplyDamageToTarget(AActor* Target)
{
    if (!DamageEffectClass)
    {
        UE_LOG(LogTemp, Error, TEXT("RPGProjectile: No DamageEffectClass set!"));
        return;
    }

    // 获取源ASC
    UAbilitySystemComponent* SourceASCPtr = SourceASC.Get();
    if (!IsValid(SourceASCPtr))
    {
        return;
    }

    // 获取目标ASC
    UAbilitySystemComponent* TargetASC =
        UAbilitySystemGlobals::GetAbilitySystemComponentFromActor(Target);

    if (!IsValid(TargetASC))
    {
        return;
    }

    // --- 创建伤害GE上下文 ---
    FGameplayEffectContextHandle Context = SourceASCPtr->MakeEffectContext();
    Context.AddSourceObject(this);  // 投射物自身为来源对象

    // --- 创建GE Spec ---
    FGameplayEffectSpecHandle SpecHandle = SourceASCPtr->MakeOutgoingSpec(
        DamageEffectClass, 1.0f, Context);

    if (SpecHandle.IsValid())
    {
        // 设置技能倍率（SetByCaller）
        SpecHandle.Data->SetSetByCallerMagnitude(
            FGameplayTag::RequestGameplayTag("Data.SkillMultiplier"),
            SkillMultiplier
        );

        // 应用GE到目标
        SourceASCPtr->ApplyGameplayEffectSpecToTarget(*SpecHandle.Data, TargetASC);
    }
}
```

```cpp
        return;
    }

    // --- 提交冷却和消耗 ---
    if (!CommitAbility(Handle, ActorInfo, ActivationInfo))
    {
        EndAbility(Handle, ActorInfo, ActivationInfo, true, true);
        return;
    }

    // --- 播放发射动画 ---
    if (AbilityMontage)
    {
        MontageTask = UAbilityTask_PlayMontageAndWait::CreatePlayMontageAndWaitProxy(
            this, NAME_None, AbilityMontage, MontagePlayRate, MontageStartSection);

        MontageTask->OnCompleted.AddDynamic(this, &UGA_FireProjectile::OnMontageCompleted);
        MontageTask->ReadyForActivation();
    }

    // --- 立即生成投射物（不等动画播完）---
    // 有些游戏会在动画的特定帧生成投射物
    // 这里为了简单，在技能激活时立即生成
    // 如果需要精确控制，可以使用WaitGameplayEvent等待动画通知
    SpawnProjectile();

    // 如果没有动画，直接结束技能
    if (!AbilityMontage)
    {
        EndAbility(Handle, ActorInfo, ActivationInfo, true, false);
    }
}

void UGA_FireProjectile::EndAbility(
    const FGameplayAbilitySpecHandle Handle,
    const FGameplayAbilityActorInfo* ActorInfo,
    const FGameplayAbilityActivationInfo ActivationInfo,
    bool bReplicateEndAbility,
    bool bWasCancelled)
{
    if (MontageTask && MontageTask->IsActive())
    {
        MontageTask->EndTask();
    }

    Super::EndAbility(Handle, ActorInfo, ActivationInfo, bReplicateEndAbility, bWasCancelled);
}

void UGA_FireProjectile::SpawnProjectile()
{
    // ================================================================
    // 第一步：计算发射位置和方向
    // ================================================================
    const FVector FireLocation = GetSourceActor()->GetActorLocation()
        + GetSourceActor()->GetActorRotation().RotateVector(SpawnOffset);

    const FVector FireDirection = GetFireDirection();
    const FRotator FireRotation = FireDirection.Rotation();

    // ================================================================
    // 第二步：生成投射物Actor
    // ================================================================
    FActorSpawnParameters SpawnParams;
    SpawnParams.Owner = GetSourceActor();            // 所有者为发射者
    SpawnParams.Instigator = Cast<APawn>(GetSourceActor());  // 发起者（用于击杀统计）
    SpawnParams.SpawnCollisionHandlingOverride =
        ESpawnActorCollisionHandlingMethod::AlwaysSpawn;  // 总是生成（即使碰撞）

    ARPGProjectile* Projectile = GetWorld()->SpawnActor<ARPGProjectile>(
        ProjectileClass,        // 投射物蓝图类
        FireLocation,           // 生成位置
        FireRotation,           // 生成方向
        SpawnParams             // 生成参数
    );

    if (IsValid(Projectile))
    {
        // ================================================================
        // 第三步：初始化投射物
        // ================================================================
        Projectile->InitializeProjectile(
            GetSourceASC(),         // 攻击者的ASC
            DamageEffectClass,      // 伤害GE类
            ProjectileSpeed,        // 飞行速度
            ProjectileMaxRange,     // 最大射程
            SkillMultiplier         // 技能倍率
        );

        UE_LOG(LogTemp, Log, TEXT("GA_FireProjectile: Projectile spawned at %s"),
            *FireLocation.ToString());
    }
}

FVector UGA_FireProjectile::GetFireDirection() const
{
    if (!bUseAimDirection)
    {
        // 使用角色朝向
        return GetSourceActor()->GetActorForwardVector();
    }

    // ================================================================
    // 瞄准模式：使用摄像机方向（准星方向）
    // ================================================================

    // 获取玩家控制器
    ACharacter* Character = Cast<ACharacter>(GetSourceActor());
    if (!IsValid(Character))
    {
        return GetSourceActor()->GetActorForwardVector();
    }

    APlayerController* PC = Cast<APlayerController>(Character->GetController());
    if (!IsValid(PC))
    {
        return GetSourceActor()->GetActorForwardVector();
    }

    // 获取摄像机位置和方向
    FVector CameraLocation;
    FRotator CameraRotation;
    PC->GetPlayerViewPoint(CameraLocation, CameraRotation);

    // ================================================================
    // 进阶：使用准星射线检测
    // 如果摄像机看到地板，投射物的Z方向应该保持水平
    // 这里简化处理：直接使用摄像机朝向
    // ================================================================
    return CameraRotation.Vector();
}
```

```cpp
void UGA_Dash::ActivateAbility(
    const FGameplayAbilitySpecHandle Handle,
    const FGameplayAbilityActorInfo* ActorInfo,
    const FGameplayAbilityActivationInfo ActivationInfo,
    const FGameplayEventData* TriggerEventData)
{
    Super::ActivateAbility(Handle, ActorInfo, ActivationInfo, TriggerEventData);

    // --- 提交冷却和消耗 ---
    if (!CommitAbility(Handle, ActorInfo, ActivationInfo))
    {
        EndAbility(Handle, ActorInfo, ActivationInfo, true, true);
        return;
    }

    // --- 获取角色 ---
    ACharacter* Character = Cast<ACharacter>(GetSourceActor());
    if (!IsValid(Character))
    {
        EndAbility(Handle, ActorInfo, ActivationInfo, true, true);
        return;
    }

    // ================================================================
    // 添加无敌标签（如果启用）
    // ================================================================
    if (bInvincibleDuringDash && InvincibleTag.IsValid())
    {
        GetSourceASC()->AddLooseGameplayTag(InvincibleTag);
        UE_LOG(LogTemp, Log, TEXT("GA_Dash: Invincibility activated"));
    }

    // ================================================================
    // 计算冲刺目标位置
    // ================================================================
    // InputPressed 的方向可以从当前输入方向获取
    // 更简单的方式：使用角色当前面朝方向
    DashStartLocation = Character->GetActorLocation();

    // 获取冲刺方向（优先使用输入方向，否则使用角色朝向）
    FVector DashDirection = Character->GetLastMovementInputVector();
    if (DashDirection.IsNearlyZero())
    {
        DashDirection = Character->GetActorForwardVector();
    }
    DashDirection.Normalize();

    DashTargetLocation = DashStartLocation + DashDirection * DashDistance;

    // ================================================================
    // 安全检查：目标位置是否可达
    // ================================================================
    // 使用Sweep检查目标位置是否有障碍物
    FHitResult HitResult;
    FCollisionQueryParams QueryParams;
    QueryParams.AddIgnoredActor(Character);

    bool bHit = GetWorld()->LineTraceSingleByChannel(
        HitResult,
        DashStartLocation,
        DashTargetLocation,
        ECC_WorldStatic,
        QueryParams
    );

    if (bHit)
    {
        // 如果撞墙了，缩短冲刺距离到碰撞点之前
        DashTargetLocation = HitResult.Location
            - DashDirection * Character->GetCapsuleComponent()->GetScaledCapsuleRadius();
        UE_LOG(LogTemp, Log, TEXT("GA_Dash: Hit obstacle, shortened dash distance"));
    }

    // ================================================================
    // 启动冲刺计时器
    // ================================================================
    // 使用Timer每帧更新位置（平滑插值）
    DashStartTime = GetWorld()->GetTimeSeconds();

    GetWorld()->GetTimerManager().SetTimer(
        DashTimerHandle,
        this,                                   // 回调对象
        &UGA_Dash::UpdateDashPosition,          // 回调函数
        0.016f,                                 // 更新间隔（约60FPS）
        true                                    // 循环
    );

    // 设置技能结束计时器
    // 冲刺持续时间结束后自动调用EndAbility
    FTimerHandle EndTimerHandle;
    GetWorld()->GetTimerManager().SetTimer(
        EndTimerHandle,
        [this, Handle, ActorInfo, ActivationInfo]()
        {
            // Lambda捕获必要参数，调用EndAbility
            EndAbility(Handle, ActorInfo, ActivationInfo, true, false);
        },
        DashDuration,   // 延迟时间
        false           // 不循环，只执行一次
    );

    UE_LOG(LogTemp, Log, TEXT("GA_Dash: Dash started from %s to %s"),
        *DashStartLocation.ToString(), *DashTargetLocation.ToString());
}

void UGA_Dash::UpdateDashPosition()
{
    ACharacter* Character = Cast<ACharacter>(GetSourceActor());
    if (!IsValid(Character))
    {
        return;
    }

    // ================================================================
    // 计算当前进度（0.0 ~ 1.0）
    // ================================================================
    const float CurrentTime = GetWorld()->GetTimeSeconds();
    const float Elapsed = CurrentTime - DashStartTime;
    float Progress = FMath::Clamp(Elapsed / DashDuration, 0.0f, 1.0f);

    // ================================================================
    // 应用速度曲线（如果有）
    // ================================================================
    if (DashSpeedCurve)
    {
        // 曲线定义速度随时间的变化
        // 例如：开始快→中间慢→结束快（先加速后减速）
        const float CurveValue = DashSpeedCurve->GetFloatValue(Progress);

        // 计算从起点到当前进度经过的距离
        const float TotalDistanceSoFar = DashDistance * CurveValue;
        const FVector DashDirection = (DashTargetLocation - DashStartLocation).GetSafeNormal();
        const FVector NewLocation = DashStartLocation + DashDirection * TotalDistanceSoFar;

        // 直接设置位置（闪现风格，忽略物理）
        // ⚠️ 注意：这种方式绕过了碰撞检测
        // 如果需要碰撞，改用CharacterMovement的AddImpulse或LaunchCharacter
        Character->SetActorLocation(NewLocation, false, nullptr, ETeleportType::None);
    }
    else
    {
        // 无曲线：线性插值（匀速移动）
        const FVector NewLocation = FMath::Lerp(
            DashStartLocation,
            DashTargetLocation,
            Progress
        );
        Character->SetActorLocation(NewLocation, false, nullptr, ETeleportType::None);
    }

    // 可选：在移动过程中旋转角色朝向移动方向
    // const FVector MoveDirection = (DashTargetLocation - DashStartLocation).GetSafeNormal();
    // Character->SetActorRotation(MoveDirection.Rotation());
}

void UGA_Dash::EndAbility(
    const FGameplayAbilitySpecHandle Handle,
    const FGameplayAbilityActorInfo* ActorInfo,
    const FGameplayAbilityActivationInfo ActivationInfo,
    bool bReplicateEndAbility,
    bool bWasCancelled)
{
    // ================================================================
    // 清理冲刺计时器
    // ================================================================
    if (GetWorld())
    {
        GetWorld()->GetTimerManager().ClearTimer(DashTimerHandle);
    }

    // ================================================================
    // 移除无敌标签
    // ================================================================
    if (bInvincibleDuringDash && InvincibleTag.IsValid())
    {
        if (UAbilitySystemComponent* ASC = GetSourceASC())
        {
            ASC->RemoveLooseGameplayTag(InvincibleTag);

            UE_LOG(LogTemp, Log, TEXT("GA_Dash: Invincibility removed"));
        }
    }

    Super::EndAbility(Handle, ActorInfo, ActivationInfo, bReplicateEndAbility, bWasCancelled);
}
```

```cpp
void UGA_AreaHeal::EndAbility(
    const FGameplayAbilitySpecHandle Handle,
    const FGameplayAbilityActorInfo* ActorInfo,
    const FGameplayAbilityActivationInfo ActivationInfo,
    bool bReplicateEndAbility,
    bool bWasCancelled)
{
    // 清理计时器
    if (GetWorld())
    {
        GetWorld()->GetTimerManager().ClearTimer(PulseTimerHandle);
    }

    if (MontageTask && MontageTask->IsActive())
    {
        MontageTask->EndTask();
    }

    Super::EndAbility(Handle, ActorInfo, ActivationInfo, bReplicateEndAbility, bWasCancelled);
}

void UGA_AreaHeal::PerformHealPulse()
{
    // ================================================================
    // 检查脉冲次数
    // ================================================================
    CurrentPulse++;

    if (CurrentPulse > PulseCount)
    {
        // 所有脉冲完成，结束技能
        EndAbility(CurrentSpecHandle, CurrentActorInfo, CurrentActivationInfo, true, false);
        return;
    }

    UE_LOG(LogTemp, Log, TEXT("GA_AreaHeal: Heal Pulse %d/%d"), CurrentPulse, PulseCount);

    // ================================================================
    // 获取范围内的有效目标
    // ================================================================
    TArray<AActor*> Targets = GetValidTargetsInRange();

    UE_LOG(LogTemp, Log, TEXT("GA_AreaHeal: Found %d target(s) in range"), Targets.Num());

    // ================================================================
    // 对每个目标应用治疗
    // ================================================================
    for (AActor* Target : Targets)
    {
        ApplyHealToTarget(Target);
    }

    // ================================================================
    // 最后一脉冲后结束技能
    // ================================================================
    if (CurrentPulse >= PulseCount)
    {
        EndAbility(CurrentSpecHandle, CurrentActorInfo, CurrentActivationInfo, true, false);
    }
}

TArray<AActor*> UGA_AreaHeal::GetValidTargetsInRange() const
{
    TArray<AActor*> Result;

    AActor* SourceActor = GetSourceActor();
    if (!IsValid(SourceActor))
    {
        return Result;
    }

    // ================================================================
    // 球形扫描获取范围内的所有Pawn
    // ================================================================
    TArray<TEnumAsByte<EObjectTypeQuery>> ObjectTypes;
    ObjectTypes.Add(UEngineTypes::ConvertToObjectType(ECC_Pawn));

    TArray<AActor*> ActorsToIgnore;

    TArray<FOverlapResult> OverlapResults;

    bool bOverlap = UKismetSystemLibrary::SphereOverlapActors(
        SourceActor->GetWorld(),
        SourceActor->GetActorLocation(),
        HealRadius,
        ObjectTypes,
        APawn::StaticClass(),
        ActorsToIgnore,
        OverlapResults
    );

    if (!bOverlap)
    {
        return Result;  // 范围内没有目标
    }

    for (const FOverlapResult& Overlap : OverlapResults)
    {
        AActor* HitActor = Overlap.GetActor();
        if (!IsValid(HitActor))
        {
            continue;
        }

        // --- 是否排除自己？ ---
        if (!bHealSelf && HitActor == SourceActor)
        {
            continue;
        }

        // --- 是否只治疗友方？ ---
        if (bHealAlliesOnly)
        {
            // 在实际项目中检查Team/Faction
            // 这里简化为：有ASC的角色都可以被治疗
        }

        // --- 目标必须有ASC（才能应用治疗GE）---
        UAbilitySystemComponent* TargetASC =
            UAbilitySystemGlobals::GetAbilitySystemComponentFromActor(HitActor);

        if (!IsValid(TargetASC))
        {
            continue;  // 目标没有GAS（可能是装饰NPC）
        }

        // --- 检查目标是否已死亡 ---
        if (TargetASC->HasMatchingGameplayTag(
            FGameplayTag::RequestGameplayTag("State.Dead")))
        {
            continue;  // 不治疗死人
        }

        Result.Add(HitActor);
    }

    return Result;
}

void UGA_AreaHeal::ApplyHealToTarget(AActor* Target)
{
    if (!IsValid(HealEffectClass))
    {
        UE_LOG(LogTemp, Error, TEXT("GA_AreaHeal: No HealEffectClass set!"));
        return;
    }

    // 获取目标的ASC
    UAbilitySystemComponent* TargetASC =
        UAbilitySystemGlobals::GetAbilitySystemComponentFromActor(Target);

    if (!IsValid(TargetASC))
    {
        return;
    }

    UAbilitySystemComponent* SourceASC = GetSourceASC();
    if (!IsValid(SourceASC))
    {
        return;
    }

    // --- 创建治疗GE上下文 ---
    FGameplayEffectContextHandle Context = SourceASC->MakeEffectContext();
    Context.AddSourceObject(this);

    // --- 创建GE Spec ---
    FGameplayEffectSpecHandle SpecHandle = SourceASC->MakeOutgoingSpec(
        HealEffectClass,
        GetAbilityLevel(),
        Context
    );

    if (SpecHandle.IsValid())
    {
        // 设置治疗倍率
        SpecHandle.Data->SetSetByCallerMagnitude(
            FGameplayTag::RequestGameplayTag("Data.HealMultiplier"),
            HealMultiplier
        );

        // 应用GE到目标
        SourceASC->ApplyGameplayEffectSpecToTarget(*SpecHandle.Data, TargetASC);

        // 可选：播放治疗特效（通过GameplayCue）
    }
}
```

## 2.8 技能输入绑定完整配置指南

### 2.8.1 在编辑器中创建增强输入资产

**步骤1：创建InputAction资产**

在Content Browser中：

1. 右键 → Input → Input Action
2. 命名规则：`IA_技能名`，例如：
   - `IA_Attack_Light` — 轻攻击
   - `IA_Skill_Fireball` — 火球术
   - `IA_Skill_Dash` — 闪现
   - `IA_Skill_Heal` — 范围治疗

**步骤2：创建InputMappingContext资产**

1. 右键 → Input → Input Mapping Context
2. 命名为 `IMC_Combat`
3. 打开，添加Mappings：
   - `IA_Attack_Light` → Left Mouse Button
   - `IA_Skill_Fireball` → Q Key
   - `IA_Skill_Dash` → Space Bar (双击或配合Shift)
   - `IA_Skill_Heal` → E Key

**步骤3：配置InputBindingComponent**

在角色蓝图中：

1. 选择 `RPGInputBindingComponent`
2. 在Details面板中的 `Input Bindings` 数组添加条目：
   - 条目1：`InputAction=IA_Attack_Light`, `InputTag_Pressed=Input.Attack.Light`
   - 条目2：`InputAction=IA_Skill_Fireball`, `InputTag_Pressed=Input.Skill.Fireball`
   - 条目3：`InputAction=IA_Skill_Dash`, `InputTag_Pressed=Input.Movement.Dash`
   - 条目4：`InputAction=IA_Skill_Heal`, `InputTag_Pressed=Input.Skill.Heal`

**步骤4：在GA中设置InputTag**

每个GA蓝图的Details面板中：

- `GA_MeleeAttack` → `InputTag` = `Input.Attack.Light`
- `GA_FireProjectile` → `InputTag` = `Input.Skill.Fireball`
- `GA_Dash` → `InputTag` = `Input.Movement.Dash`
- `GA_AreaHeal` → `InputTag` = `Input.Skill.Heal`

**步骤5：授予技能给角色**

在 `ARPGCharacter::BeginPlay` 中（或在蓝图 `Event BeginPlay` 中）：
