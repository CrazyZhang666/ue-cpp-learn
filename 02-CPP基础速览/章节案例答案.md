## 练习1：按稀有度排序

### 实现代码

在 `CInventory` 类中添加以下方法：

```cpp
// 按稀有度排序（传说 → 史诗 → 稀有 → 不凡 → 普通）
void SortByRarity()
{
    // sort 是标准库算法，需要 #include <algorithm>
    // 第三个参数是"比较函数"，返回true表示第一个参数应该排在前面
    sort(Items.begin(), Items.end(),
        [](const FItem& A, const FItem& B)  // Lambda表达式（匿名函数）
        {
            // 枚举底层是整数，数值越大越稀有，所以用 > 降序排列
            return static_cast<int>(A.Rarity) > static_cast<int>(B.Rarity);
        });

    cout << "【排序】背包已按稀有度重新排列" << endl;
}
```

### 使用示例

在 `main()` 函数末尾添加：

```cpp
cout << endl << "=== 测试6：按稀有度排序 ===" << endl;
Backpack.SortByRarity();
Backpack.Display();
```

### 预期输出

```
【排序】背包已按稀有度重新排列

╔══════════════════════════════════════╗
║        背 包 物 品 列 表              ║
╠══════════════════════════════════════╣
║ [1] 龙鳞 x1 (传说) - 2.0kg          ║
║ [2] 生命药水 x1 (普通) - 0.5kg      ║
║ [3] 法力药水 x1 (普通) - 0.3kg      ║
╚══════════════════════════════════════╝
```

### 知识点说明

| 概念                  | 说明                                             |
| --------------------- | ------------------------------------------------ |
| `sort()`              | 标准库排序算法，时间复杂度 O(n log n)            |
| Lambda                | `[](){}` 是C++11引入的匿名函数，方便写一次性逻辑 |
| `static_cast`         | 安全的类型转换，把枚举转成整数比较               |
| `Items.begin()/end()` | vector 的起始/结束迭代器，sort 需要它们确定范围  |

## 练习2：使用物品

### 实现代码

在 `CInventory` 类中添加以下方法：

```cpp
// 使用物品：堆叠数-1，归零自动删除
bool UseItem(const string& ItemName)
{
    // 遍历查找物品
    for (int i = 0; i < Items.size(); i++)
    {
        if (Items[i].Name == ItemName)
        {
            // 找到物品，减少堆叠数量
            Items[i].CurrentStack--;
            cout << "【使用】使用了 " << ItemName
                 << "，剩余: " << Items[i].CurrentStack << endl;

            // 如果堆叠归零，从背包中移除
            if (Items[i].CurrentStack <= 0)
            {
                cout << "【提示】" << ItemName << " 已用完，从背包移除" << endl;
                Items.erase(Items.begin() + i);
            }
            return true;  // 使用成功
        }
    }

    cout << "【失败】背包中没有 " << ItemName << "，无法使用" << endl;
    return false;  // 使用失败（物品不存在）
}
```

### 使用示例

在 `main()` 函数末尾添加：

```cpp
cout << endl << "=== 测试7：使用物品 ===" << endl;
Backpack.UseItem("生命药水");  // 使用1瓶，从1瓶变为0瓶，自动删除
Backpack.UseItem("龙鳞");      // 使用龙鳞
Backpack.UseItem("神剑");      // 使用不存在的物品，失败

Backpack.Display();
```

### 预期输出

```
=== 测试7：使用物品 ===
【使用】使用了 生命药水，剩余: 0
【提示】生命药水 已用完，从背包移除
【使用】使用了 龙鳞，剩余: 0
【提示】龙鳞 已用完，从背包移除
【失败】背包中没有 神剑，无法使用
```

### 知识点说明

| 概念       | 说明                                                               |
| ---------- | ------------------------------------------------------------------ |
| 递减运算符 | `CurrentStack--` 先使用值再减1（后置递减）                         |
| 条件删除   | 先减少数量，再判断是否需要从容器中移除                             |
| 索引遍历   | 这里用 `for (int i = 0; ...)` 而不是范围for，因为 `erase` 需要索引 |

## 练习3：UE版本改写

### 完整UE代码

```cpp
// InventoryComponent.h
#pragma once

#include "CoreMinimal.h"
#include "Components/ActorComponent.h"
#include "InventoryComponent.generated.h"

// 物品稀有度枚举
UENUM(BlueprintType)  // 可以在蓝图中使用
enum class EItemRarity : uint8
{
    Common      UMETA(DisplayName = "普通"),
    Uncommon    UMETA(DisplayName = "不凡"),
    Rare        UMETA(DisplayName = "稀有"),
    Epic        UMETA(DisplayName = "史诗"),
    Legendary   UMETA(DisplayName = "传说")
};

// 物品结构体
USTRUCT(BlueprintType)
struct FItem
{
    GENERATED_BODY()

    UPROPERTY()  // UE反射标记：成员变量
    FString Name;

    UPROPERTY()
    FString Description;

    UPROPERTY()
    EItemRarity Rarity;

    UPROPERTY()
    float Weight;

    UPROPERTY()
    int32 MaxStack;

    UPROPERTY()
    int32 CurrentStack;

    // 构造函数
    FItem()
        : Weight(0.0f)
        , MaxStack(1)
        , CurrentStack(1)
    {}

    FItem(FString InName, FString InDesc, EItemRarity InRarity,
          float InWeight, int32 InMaxStack)
        : Name(InName)
        , Description(InDesc)
        , Rarity(InRarity)
        , Weight(InWeight)
        , MaxStack(InMaxStack)
        , CurrentStack(1)
    {}

    float GetTotalWeight() const
    {
        return Weight * static_cast<float>(CurrentStack);
    }

    bool CanStack() const
    {
        return CurrentStack < MaxStack;
    }

    int32 AddToStack(int32 Amount)
    {
        int32 CanAdd = MaxStack - CurrentStack;
        int32 ActualAdd = FMath::Min(Amount, CanAdd);  // UE的Min函数
        CurrentStack += ActualAdd;
        return ActualAdd;
    }
};

UCLASS(ClassGroup = (Custom), meta = (BlueprintSpawnableComponent))
class MYPROJECT_API UInventoryComponent : public UActorComponent
{
    GENERATED_BODY()

public:
    UInventoryComponent();

    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Inventory")
    float MaxWeight;

    // 添加物品
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    bool AddItem(const FItem& NewItem);

    // 删除物品
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    bool RemoveItem(const FString& ItemName, int32 Amount = 1);

    // 查找物品
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    FItem* FindItem(const FString& ItemName);

    // 使用物品
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    bool UseItem(const FString& ItemName);

    // 按稀有度排序
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    void SortByRarity();

    // 获取总重量
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    float GetTotalWeight() const;

    // 获取物品种类数量
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    int32 GetItemCount() const;

    // 显示背包内容（调试用）
    UFUNCTION(BlueprintCallable, Category = "Inventory")
    void Display() const;

protected:
    UPROPERTY()
    TArray<FItem> Items;  // UE的动态数组，替代 std::vector
};
```

```cpp
// InventoryComponent.cpp
#include "InventoryComponent.h"

UInventoryComponent::UInventoryComponent()
{
    MaxWeight = 100.0f;
}

bool UInventoryComponent::AddItem(const FItem& NewItem)
{
    float CurrentWeight = GetTotalWeight();
    float NewItemWeight = NewItem.GetTotalWeight();

    if (CurrentWeight + NewItemWeight > MaxWeight)
    {
        UE_LOG(LogTemp, Warning, TEXT("背包已满！当前负重: %.1fkg, 最大负重: %.1fkg"),
            CurrentWeight, MaxWeight);
        return false;
    }

    // 尝试堆叠
    for (FItem& ExistingItem : Items)
    {
        if (ExistingItem.Name == NewItem.Name)
        {
            if (ExistingItem.CanStack())
            {
                int32 Added = ExistingItem.AddToStack(NewItem.CurrentStack);
                UE_LOG(LogTemp, Log, TEXT("%s 堆叠+%d，当前: %d/%d"),
                    *ExistingItem.Name, Added, ExistingItem.CurrentStack, ExistingItem.MaxStack);
                return true;
            }
            else
            {
                UE_LOG(LogTemp, Warning, TEXT("%s 已达最大堆叠数"), *ExistingItem.Name);
                return false;
            }
        }
    }

    // 添加新物品
    Items.Add(NewItem);
    UE_LOG(LogTemp, Log, TEXT("获得 %s"), *NewItem.Name);
    return true;
}

bool UInventoryComponent::RemoveItem(const FString& ItemName, int32 Amount)
{
    for (int32 i = 0; i < Items.Num(); i++)  // Num() = size()
    {
        if (Items[i].Name == ItemName)
        {
            if (Items[i].CurrentStack <= Amount)
            {
                UE_LOG(LogTemp, Log, TEXT("移除 %s x%d"),
                    *Items[i].Name, Items[i].CurrentStack);
                Items.RemoveAt(i);  // 替代 erase()
            }
            else
            {
                Items[i].CurrentStack -= Amount;
                UE_LOG(LogTemp, Log, TEXT("%s -%d，剩余: %d"),
                    *Items[i].Name, Amount, Items[i].CurrentStack);
            }
            return true;
        }
    }

    UE_LOG(LogTemp, Warning, TEXT("背包中没有 %s"), *ItemName);
    return false;
}

FItem* UInventoryComponent::FindItem(const FString& ItemName)
{
    for (FItem& Item : Items)
    {
        if (Item.Name == ItemName)
        {
            return &Item;
        }
    }
    return nullptr;
}

bool UInventoryComponent::UseItem(const FString& ItemName)
{
    for (int32 i = 0; i < Items.Num(); i++)
    {
        if (Items[i].Name == ItemName)
        {
            Items[i].CurrentStack--;
            UE_LOG(LogTemp, Log, TEXT("使用 %s，剩余: %d"),
                *ItemName, Items[i].CurrentStack);

            if (Items[i].CurrentStack <= 0)
            {
                UE_LOG(LogTemp, Log, TEXT("%s 已用完"), *ItemName);
                Items.RemoveAt(i);
            }
            return true;
        }
    }

    UE_LOG(LogTemp, Warning, TEXT("没有 %s，无法使用"), *ItemName);
    return false;
}

void UInventoryComponent::SortByRarity()
{
    // UE的排序算法
    Items.Sort([](const FItem& A, const FItem& B)
    {
        return static_cast<uint8>(A.Rarity) > static_cast<uint8>(B.Rarity);
    });

    UE_LOG(LogTemp, Log, TEXT("背包已按稀有度排序"));
}

float UInventoryComponent::GetTotalWeight() const
{
    float Total = 0.0f;
    for (const FItem& Item : Items)
    {
        Total += Item.GetTotalWeight();
    }
    return Total;
}

int32 UInventoryComponent::GetItemCount() const
{
    return Items.Num();
}

void UInventoryComponent::Display() const
{
    UE_LOG(LogTemp, Log, TEXT("========== 背包内容 =========="));

    if (Items.Num() == 0)
    {
        UE_LOG(LogTemp, Log, TEXT("（背包空空如也）"));
    }
    else
    {
        for (int32 i = 0; i < Items.Num(); i++)
        {
            const FItem& Item = Items[i];
            UE_LOG(LogTemp, Log, TEXT("[%d] %s x%d - %.1fkg"),
                i + 1, *Item.Name, Item.CurrentStack, Item.GetTotalWeight());
        }
    }

    UE_LOG(LogTemp, Log, TEXT("总负重: %.1fkg / %.1fkg"),
        GetTotalWeight(), MaxWeight);
    UE_LOG(LogTemp, Log, TEXT("============================="));
}
```

### 控制台版本 vs UE版本 对照表

| 标准C++             | UE C++                     | 说明                      |
| ------------------- | -------------------------- | ------------------------- |
| `#include <vector>` | `#include "CoreMinimal.h"` | UE用PCH预编译头           |
| `std::vector`       | `TArray`                   | UE的动态数组，功能更丰富  |
| `std::string`       | `FString`                  | UE的字符串类，支持UTF-16  |
| `std::cout`         | `UE_LOG`                   | 输出到日志和输出窗口      |
| `.push_back()`      | `.Add()`                   | 添加元素                  |
| `.size()`           | `.Num()`                   | 获取元素数量              |
| `.erase()`          | `.RemoveAt()`              | 删除指定位置元素          |
| `static_cast<int>`  | `static_cast<int32>`       | UE使用固定宽度整数类型    |
| `std::min`          | `FMath::Min`               | UE的数学库                |
| `sort()`            | `.Sort()`                  | TArray自带排序方法        |
| `string`            | `FString`                  | UE字符串前要加 `*` 解引用 |

### 关键差异说明

1. **反射系统**：UE中所有需要在蓝图中访问的类型都需要 `USTRUCT()` / `UCLASS()` / `UPROPERTY()` / `UFUNCTION()` 宏
2. **字符串操作**：`FString` 前面加 `*` 才能拿到 `TCHAR*` 给日志用（如 `*Item.Name`）
3. **整数类型**：UE使用 `int32` 而不是 `int`，确保跨平台字节数一致
4. **头文件包含顺序**：`.generated.h` 必须是最后一个包含的头文件
5. **日志格式**：`UE_LOG(Category, Verbosity, TEXT("..."))`，字符串必须用 `TEXT()` 宏包裹
