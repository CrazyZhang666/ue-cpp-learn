# 第4章案例：可旋转巡逻的AI警卫

> **难度**：⭐⭐⭐（综合实战级）
> **预计时间**：90分钟
> **目标**：综合运用Actor生命周期、Component体系、生成与销毁等全部章节知识，实现一个完整的AI巡逻警卫。

---

## 案例目标

创建一个 `APatrolGuard` AI角色，具备以下能力：

```
┌──────────── 巡逻警卫 AI ────────────┐
│                                     │
│  1. 沿路径点自动巡逻                │
│    路径点 → 路径点 → 路径点 → ...   │
│                                     │
│  2. 视野检测                         │
│    使用 USphereComponent 探测玩家   │
│                                     │
│  3. 发现玩家后追击                  │
│    丢失视野3秒后回到巡逻路线        │
│                                     │
│  4. 面向移动方向                    │
│    使用FRotator::RoundInterpTo      │
│    平滑旋转面向移动方向             │
│                                     │
└─────────────────────────────────────┘
```

## 第一步：创建巡逻路径点Actor

首先需要一个路径点，让警卫知道该走向哪里。

```cpp
// ========== PatrolPoint.h ==========
#pragma once

#include "CoreMinimal.h"
#include "Engine/TargetPoint.h"  // TargetPoint 是 UE 内置的标记点类
#include "PatrolPoint.generated.h"

/**
 * 巡逻路径点
 *
 * 继承自 ATargetPoint（UE内置的标记Actor）
 * TargetPoint 继承自 AActor，自带一个箭头显示方向
 * 直接拖入关卡即可使用
 *
 * 为什么用TargetPoint？
 * - 自带可视化的箭头（编辑器中可以看到）
 * - 轻量（没有多余逻辑）
 * - 引擎内置，不用自己写
 */
UCLASS()
class MYGAME_API APatrolPoint : public ATargetPoint
{
    GENERATED_BODY()

public:
    APatrolPoint();

    /**
     * 警卫到达此路径点后停留的时间（秒）
     * 0 = 不停留，直接走向下一个
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Patrol",
              meta = (DisplayName = "停留时间", ClampMin = "0.0"))
    float WaitTime = 0.0f;

    /**
     * 到达此路径点的距离阈值（厘米）
     * 警卫距离此路径点小于此值时，认为"到达"
     * 防止因为浮点精度永远到不了
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Patrol",
              meta = (DisplayName = "到达距离阈值", ClampMin = "10.0"))
    float ArrivalRadius = 100.0f;  // 1米内就算到达
};

// ========== PatrolPoint.cpp ==========
#include "PatrolPoint.h"

APatrolPoint::APatrolPoint()
{
    // 路径点不需要Tick
    PrimaryActorTick.bCanEverTick = false;

    // 路径点通常不可见（只在编辑器中显示箭头）
    // 如果想让它在游戏中可见用于调试，这里需要额外设置
}
```

## 第二步：创建巡逻警卫Actor（核心）

```cpp
// ========== PatrolGuard.h ==========
#pragma once

#include "CoreMinimal.h"
#include "GameFramework/Character.h"  // Character = 人形角色（带骨骼、动画、移动）
#include "PatrolGuard.generated.h"

class USphereComponent;     // 用于视野检测
class APatrolPoint;         // 我们刚才创建的巡逻点

/**
 * 巡逻警卫AI — 本章的综合实战案例
 *
 * 继承关系：
 *   AActor → APawn → ACharacter → APatrolGuard
 *              ↑        ↑           ↑
 *          可被控制   人形角色    我们的警卫
 *
 * 为什么用ACharacter？
 * - 自带CapsuleComponent（胶囊碰撞体）
 * - 自带CharacterMovementComponent（角色移动）
 * - 自带SkeletalMeshComponent（骨骼模型，可以做动画）
 * - 支持Walking（走路移动模式，自动处理地面/重力）
 */
UCLASS()
class MYGAME_API APatrolGuard : public ACharacter
{
    GENERATED_BODY()

public:
    APatrolGuard();

protected:
    // ===== 生命周期 =====
    virtual void BeginPlay() override;
    virtual void Tick(float DeltaTime) override;

public:
    // ===== 组件 =====

    /** 视野检测球 — 当玩家进入这个范围，警卫发现玩家 */
    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "Guard | Components")
    USphereComponent* VisionSphere;

    // ===== 巡逻配置 =====

    /**
     * 巡逻路径点数组
     * 在编辑器中拖入关卡中的 APatrolPoint
     * 警卫会按数组顺序依次巡逻
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Guard | Patrol",
              meta = (DisplayName = "巡逻路径点"))
    TArray<APatrolPoint*> PatrolPoints;

    /**
     * 巡逻移动速度（cm/s）
     * 默认150cm/s = 1.5m/s ≈ 慢走速度
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Guard | Patrol",
              meta = (DisplayName = "巡逻速度"))
    float PatrolSpeed = 150.0f;

    /**
     * 追击移动速度（cm/s）
     * 默认450cm/s = 4.5m/s ≈ 快跑
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Guard | Patrol",
              meta = (DisplayName = "追击速度"))
    float ChaseSpeed = 450.0f;

    /**
     * 视野半径（厘米）
     * 默认800cm = 8米
     * 玩家进入此范围即被发现
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Guard | Vision",
              meta = (DisplayName = "视野半径"))
    float VisionRadius = 800.0f;

    /**
     * 丢失目标后的记忆时间（秒）
     * 玩家离开视野后，警卫仍会追击这段时间
     * 超过时间仍看不到，则回到巡逻路线
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Guard | Vision",
              meta = (DisplayName = "丢失视野记忆时间", ClampMin = "0.0"))
    float LostTargetMemoryTime = 3.0f;

    /**
     * 旋转平滑速度
     * 值越大旋转越快（单位：度/秒的倍率）
     * 控制转身的丝滑程度
     */
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Guard | Movement",
              meta = (DisplayName = "旋转平滑速度"))
    float TurnSmoothSpeed = 5.0f;

    // ===== 运行时状态 =====

    /** 当前巡逻到的路径点索引 */
    int32 CurrentPatrolIndex = 0;

    /** 当前追击的目标（如果为nullptr，则在巡逻） */
    AActor* CurrentTarget = nullptr;

    /** 玩家离开视野后的计时器（累计超过LostTargetMemoryTime则放弃追击） */
    float TimeSinceLostTarget = 0.0f;

    // ===== 巡逻行为 =====

    /**
     * 巡逻逻辑：走向当前路径点
     * @param DeltaTime — 帧间隔时间
     */
    void PatrolBehavior(float DeltaTime);

    /**
     * 追击逻辑：走向玩家
     * @param DeltaTime — 帧间隔时间
     */
    void ChaseBehavior(float DeltaTime);

    /**
     * 移动到下一个巡逻点
     */
    void GoToNextPatrolPoint();

    /**
     * 开始追击目标
     * @param Target — 要追击的Actor（通常是玩家）
     */
    void StartChasing(AActor* Target);

    /**
     * 停止追击，回到巡逻路线
     */
    void StopChasing();

    /**
     * 平滑旋转面向目标位置
     * @param TargetLocation — 目标世界坐标
     * @param DeltaTime — 帧间隔时间
     */
    void SmoothLookAt(const FVector& TargetLocation, float DeltaTime);

    // ===== 碰撞回调 =====

    /** 玩家进入视野 */
    UFUNCTION()
    void OnPlayerEnterVision(UPrimitiveComponent* OverlappedComp, AActor* OtherActor,
                             UPrimitiveComponent* OtherComp, int32 OtherBodyIndex,
                             bool bFromSweep, const FHitResult& SweepResult);

    /** 玩家离开视野 */
    UFUNCTION()
    void OnPlayerExitVision(UPrimitiveComponent* OverlappedComp, AActor* OtherActor,
                            UPrimitiveComponent* OtherComp, int32 OtherBodyIndex);
};
```

## 第三步：实现巡逻警卫

```cpp
// ========== PatrolGuard.cpp ==========
#include "PatrolGuard.h"
#include "PatrolPoint.h"           // 我们的巡逻点类
#include "Components/SphereComponent.h"
#include "GameFramework/CharacterMovementComponent.h"
#include "Kismet/GameplayStatics.h" // GetPlayerCharacter 等工具函数
#include "Engine/World.h"

// =========================================
// 构造函数
// =========================================
APatrolGuard::APatrolGuard()
{
    // ----- 1. 开启Tick（警卫需要每帧巡逻/追击）-----
    PrimaryActorTick.bCanEverTick = true;

    // ----- 2. 创建视野检测球 -----
    VisionSphere = CreateDefaultSubobject<USphereComponent>(TEXT("VisionSphere"));
    //                 ↑
    // CreateDefaultSubobject: 在构造函数中创建组件
    // 参数 "VisionSphere" 是这个组件的内部名称（用于序列化标识）

    VisionSphere->SetupAttachment(RootComponent);
    //       ↑ 挂载到ACharacter自带的CapsuleComponent（RootComponent）

    VisionSphere->SetSphereRadius(VisionRadius);
    //       ↑ 设置球的半径 = 视野半径

    // ----- 3. 配置视野球的碰撞 -----
    VisionSphere->SetCollisionEnabled(ECollisionEnabled::QueryOnly);
    //       ↑ 只做碰撞查询（检测重叠），不参与物理模拟

    VisionSphere->SetCollisionResponseToAllChannels(ECR_Ignore);
    //       ↑ 先忽略所有碰撞通道

    VisionSphere->SetCollisionResponseToChannel(ECC_Pawn, ECR_Overlap);
    //       ↑ 只对Pawn通道产生Overlap（重叠）事件
    // 这样只有Pawn（玩家、其他角色）进入球体才触发回调
    // 墙壁、地板等不会触发（节省性能）

    // ----- 4. 配置角色移动组件（继承自ACharacter）-----
    UCharacterMovementComponent* MovementComp = GetCharacterMovement();
    // ACharacter自带CharacterMovementComponent

    MovementComp->bOrientRotationToMovement = false;
    //       ↑ 不自动旋转面向移动方向
    // 我们要手动控制旋转（用SmoothLookAt），所以关掉自动旋转

    MovementComp->MaxWalkSpeed = PatrolSpeed;
    //       ↑ 初始设为巡逻速度（慢走）

    MovementComp->bUseControllerDesiredRotation = false;
    //       ↑ 不使用控制器的期望旋转
    // 因为这是AI，没有玩家控制器
}

// =========================================
// BeginPlay — 游戏开始的入口
// =========================================
void APatrolGuard::BeginPlay()
{
    // ----- 1. 永远先调用父类 -----
    Super::BeginPlay();
    //       ↑
    // 铁律！调用Super::BeginPlay()确保ACharacter的初始化完成
    // 忘记调用会导致：移动组件未初始化、胶囊体未注册等

    // ----- 2. 绑定视野球的碰撞回调 -----
    VisionSphere->OnComponentBeginOverlap.AddDynamic(
        this, &APatrolGuard::OnPlayerEnterVision
    );
    //  ↑              ↑                          ↑
    // 碰撞组件    绑定到this（本Actor）    回调函数
    // 当有物体进入视野球 → 调用 OnPlayerEnterVision

    VisionSphere->OnComponentEndOverlap.AddDynamic(
        this, &APatrolGuard::OnPlayerExitVision
    );
    // 当物体离开视野球 → 调用 OnPlayerExitVision

    // ----- 3. 检查巡逻路径点是否配置 -----
    if (PatrolPoints.Num() == 0)
    {
        UE_LOG(LogTemp, Warning, TEXT("警告：%s 没有配置巡逻路径点！警卫将原地不动"),
               *GetName());
        return;
    }

    // ----- 4. 移除巡逻点中因为各种原因变成的nullptr -----
    PatrolPoints.RemoveAll([](APatrolPoint* Point)
    {
        return !IsValid(Point);  // 无效的路径点（被删除的）移除
    });
    //       ↑ RemoveAll + Lambda：过滤数组中不满足条件的元素

    if (PatrolPoints.Num() == 0)
    {
        UE_LOG(LogTemp, Warning, TEXT("所有巡逻路径点都无效！"));
        return;
    }

    // ----- 5. 将警卫传送到第一个巡逻点 -----
    // 这样可以确保巡逻从第一个点开始（而不是地图原点）
    APatrolPoint* FirstPoint = PatrolPoints[0];
    SetActorLocation(FirstPoint->GetActorLocation());
    //       ↑ 将Actor放到第一个路径点的位置

    // ----- 6. 开始巡逻 -----
    CurrentPatrolIndex = 0;  // 从第0个点开始
    GoToNextPatrolPoint();   // 走向第一个点（或第二个，取决于逻辑）

    UE_LOG(LogTemp, Log, TEXT("警卫 %s 开始巡逻，共 %d 个路径点"),
           *GetName(), PatrolPoints.Num());
}

// =========================================
// Tick — 每帧执行
// =========================================
void APatrolGuard::Tick(float DeltaTime)
{
    // ----- 1. 永远先调用父类 -----
    Super::Tick(DeltaTime);

    // ----- 2. 根据是否有目标，选择行为 -----
    if (CurrentTarget && IsValid(CurrentTarget))
    //     ↑                 ↑
    //  有追击目标     且目标仍然有效（未被销毁）
    {
        // ===== 追击模式 =====
        ChaseBehavior(DeltaTime);
    }
    else
    {
        // ===== 巡逻模式 =====
        PatrolBehavior(DeltaTime);
    }
    // 这就是典型的"有限状态机"（FSM）的雏形
    // 只有两个状态：巡逻 vs 追击
    // 以后可以扩展：战斗、警戒、返回等
}

// =========================================
// 巡逻逻辑
// =========================================
void APatrolGuard::PatrolBehavior(float DeltaTime)
{
    // ----- 1. 安全检查 -----
    if (PatrolPoints.Num() == 0) return;  // 没有路径点，无法巡逻

    APatrolPoint* TargetPoint = PatrolPoints[CurrentPatrolIndex];
    //       ↑ 获取当前要去的路径点

    if (!IsValid(TargetPoint))
    {
        // 当前路径点无效（可能被删了），跳到下一个
        GoToNextPatrolPoint();
        return;
    }

    // ----- 2. 平滑旋转面向目标点 -----
    SmoothLookAt(TargetPoint->GetActorLocation(), DeltaTime);
    //       ↑ 让警卫面向巡逻目标方向

    // ----- 3. 计算到目标点的距离 -----
    FVector MyLocation = GetActorLocation();
    FVector TargetLocation = TargetPoint->GetActorLocation();
    float Distance = FVector::Dist(MyLocation, TargetLocation);
    //     ↑ 三维距离（单位：厘米）

    // ----- 4. 检查是否到达路径点 -----
    if (Distance <= TargetPoint->ArrivalRadius)
    //                 ↑ 距离小于等于路径点定义的到达半径
    {
        // ===== 到达了！走到下一个点 =====
        if (TargetPoint->WaitTime > 0.0f)
        //                  ↑ 如果配置了停留时间
        {
            // 停一下再走（用Timer实现）
            // 简化版：直接跳下一个
            // 进阶版：用FTimerHandle实现延迟
        }
        GoToNextPatrolPoint();
    }
    else
    {
        // ===== 还没到，继续走 =====
        // 向前方移动（已经通过SmoothLookAt面朝目标了）
        FVector ForwardDir = GetActorForwardVector();
        //        ↑ Actor的前方方向向量（长度为1）

        // 使用CharacterMovementComponent的移动
        AddMovementInput(ForwardDir, 1.0f);
        //     ↑                  ↑
        //  方向（世界空间）    缩放值（1.0=全速）
        // AddMovementInput 是ACharacter提供的方法
        // 它会使用CharacterMovementComponent来处理移动
    }
}

// =========================================
// 追击逻辑
// =========================================
void APatrolGuard::ChaseBehavior(float DeltaTime)
{
    // ----- 1. 安全检查 -----
    if (!CurrentTarget || !IsValid(CurrentTarget))
    {
        // 目标突然失效（被销毁了）
        StopChasing();
        return;
    }

    // ----- 2. 计算与目标之间的距离 -----
    FVector MyLocation = GetActorLocation();
    FVector TargetLocation = CurrentTarget->GetActorLocation();
    float Distance = FVector::Dist(MyLocation, TargetLocation);

    // ----- 3. 检查目标是否在视野内 -----
    bool bTargetInVision = Distance <= VisionRadius;
    //     ↑ 简单判断：距离小于视野半径 = 在视野内

    // 进阶：可以加射线检测
    // 判断玩家和警卫之间是否有障碍物（墙壁等）
    // FHitResult Hit;
    // bool bHasLineOfSight = !GetWorld()->LineTraceSingleByChannel(
    //     Hit, MyLocation, TargetLocation, ECC_Visibility
    // );

    if (bTargetInVision)
    {
        // ===== 目标还在视野内 =====
        TimeSinceLostTarget = 0.0f;  // 重置丢失计时器

        // 平滑旋转面向玩家
        SmoothLookAt(TargetLocation, DeltaTime);

        // 追击移动
        FVector ForwardDir = GetActorForwardVector();
        AddMovementInput(ForwardDir, 1.0f);
    }
    else
    {
        // ===== 目标不在视野内 =====
        TimeSinceLostTarget += DeltaTime;
        //       ↑ 累计丢失时间

        if (TimeSinceLostTarget >= LostTargetMemoryTime)
        //                      ↑ 超过了记忆时间（默认3秒）
        {
            UE_LOG(LogTemp, Log, TEXT("警卫 %s 丢失目标，回到巡逻"), *GetName());
            StopChasing();
            // 回到巡逻路线
        }
        else
        {
            // 还记得目标位置，继续向最后看到的位置移动
            SmoothLookAt(TargetLocation, DeltaTime);
            FVector ForwardDir = GetActorForwardVector();
            AddMovementInput(ForwardDir, 1.0f);
        }
    }
}

// =========================================
// 走到下一个巡逻点
// =========================================
void APatrolGuard::GoToNextPatrolPoint()
{
    if (PatrolPoints.Num() == 0) return;

    // ----- 推进索引 -----
    CurrentPatrolIndex++;

    // ----- 循环（到头后回到起点）-----
    if (CurrentPatrolIndex >= PatrolPoints.Num())
    {
        CurrentPatrolIndex = 0;  // 回到第一个点，形成循环巡逻
    }
    // 如果不想要循环，可以这样：
    // if (CurrentPatrolIndex >= PatrolPoints.Num())
    // {
    //     // 反向巡逻
    //     // 或者停止巡逻
    // }

    APatrolPoint* NextPoint = PatrolPoints[CurrentPatrolIndex];
    if (IsValid(NextPoint))
    {
        UE_LOG(LogTemp, Log, TEXT("警卫走向巡逻点 %d：%s"),
               CurrentPatrolIndex, *NextPoint->GetName());
    }
}

// =========================================
// 开始追击
// =========================================
void APatrolGuard::StartChasing(AActor* Target)
{
    if (!IsValid(Target))
    {
        UE_LOG(LogTemp, Warning, TEXT("StartChasing: 目标无效！"));
        return;
    }

    // ----- 1. 设置追击目标 -----
    CurrentTarget = Target;
    TimeSinceLostTarget = 0.0f;

    // ----- 2. 切换到追击速度 -----
    UCharacterMovementComponent* MovementComp = GetCharacterMovement();
    if (MovementComp)
    {
        MovementComp->MaxWalkSpeed = ChaseSpeed;
        //       ↑ 从巡逻速度切换到追击速度
    }

    // ----- 3. 日志 -----
    UE_LOG(LogTemp, Log, TEXT("警卫 %s 开始追击 %s！速度: %.0f cm/s"),
           *GetName(), *Target->GetName(), ChaseSpeed);
}

// =========================================
// 停止追击
// =========================================
void APatrolGuard::StopChasing()
{
    // ----- 1. 清除追击目标 -----
    CurrentTarget = nullptr;
    TimeSinceLostTarget = 0.0f;

    // ----- 2. 切换回巡逻速度 -----
    UCharacterMovementComponent* MovementComp = GetCharacterMovement();
    if (MovementComp)
    {
        MovementComp->MaxWalkSpeed = PatrolSpeed;
        //       ↑ 从追击速度切回巡逻速度
    }

    // ----- 3. 找到最近的巡逻点，从那里继续 -----
    if (PatrolPoints.Num() > 0)
    {
        // 找到距离当前最近的巡逻点
        // （追击可能跑到离巡逻路线很远的地方）
        float ClosestDist = FLT_MAX;   // 初始化为最大浮点数
        int32 ClosestIndex = 0;

        for (int32 i = 0; i < PatrolPoints.Num(); i++)
        {
            if (IsValid(PatrolPoints[i]))
            {
                float Dist = FVector::Dist(
                    GetActorLocation(),
                    PatrolPoints[i]->GetActorLocation()
                );
                if (Dist < ClosestDist)
                {
                    ClosestDist = Dist;
                    ClosestIndex = i;
                }
            }
        }

        // 从最近的巡逻点继续
        CurrentPatrolIndex = ClosestIndex;
        GoToNextPatrolPoint();
    }

    UE_LOG(LogTemp, Log, TEXT("警卫 %s 停止追击，回到巡逻路线"), *GetName());
}

// =========================================
// 平滑旋转面向目标
// =========================================
void APatrolGuard::SmoothLookAt(const FVector& TargetLocation, float DeltaTime)
{
    // ----- 1. 计算从自己到目标的方向向量 -----
    FVector MyLocation = GetActorLocation();
    FVector DirectionToTarget = TargetLocation - MyLocation;
    //        ↑ 从自身指向目标的方向（向量减法）

    // 忽略Z轴（我们只关心水平面上的朝向）
    DirectionToTarget.Z = 0.0f;
    //       ↑ 设为0 = 不抬头/低头，只在水平面上旋转

    // ----- 2. 安全检查：方向是否为零向量 -----
    if (DirectionToTarget.IsNearlyZero())
    {
        return;  // 已经重合，无需旋转
    }

    // ----- 3. 将方向向量转为期望的旋转角 -----
    FRotator TargetRotation = DirectionToTarget.Rotation();
    //        ↑ FVector::Rotation() 返回一个FRotator
    // 例如：方向(1, 0, 0) → Rotation(0, 0, 0)
    //       方向(0, 1, 0) → Rotation(0, 90, 0) → 面朝Y轴

    // ----- 4. 获取当前的旋转 -----
    FRotator CurrentRotation = GetActorRotation();

    // ----- 5. 平滑插值旋转 -----
    FRotator NewRotation = FMath::RInterpTo(
        CurrentRotation,                // 当前旋转
        TargetRotation,                 // 目标旋转
        DeltaTime,                      // 帧间隔
        TurnSmoothSpeed                 // 插值速度（越大越快）
    );
    //       ↑
    // FMath::RInterpTo: Rotator的平滑插值
    // 原理：每帧把CurrentRotation向TargetRotation靠近一点
    // 靠近速度由 TurnSmoothSpeed 控制

    // ----- 6. 应用新旋转 -----
    SetActorRotation(NewRotation);
}

// =========================================
// 玩家进入视野回调
// =========================================
void APatrolGuard::OnPlayerEnterVision(UPrimitiveComponent* OverlappedComp,
                                        AActor* OtherActor,
                                        UPrimitiveComponent* OtherComp,
                                        int32 OtherBodyIndex,
                                        bool bFromSweep,
                                        const FHitResult& SweepResult)
{
    // ----- 1. 安全检查 -----
    if (!OtherActor || OtherActor == this)
    {
        return;  // 忽略自己和空指针
    }

    // ----- 2. 检查是不是玩家（通过Tag判断）-----
    // 方案A：通过Tag判断
    if (!OtherActor->ActorHasTag(FName(TEXT("Player"))))
    {
        return;  // 不是玩家，无视（可能是其他AI角色）
    }
    // 方案B：通过类判断
    // if (!OtherActor->IsA(APlayerCharacter::StaticClass()))
    // {
    //     return;
    // }

    // ----- 3. 开始追击 -----
    UE_LOG(LogTemp, Log, TEXT("警卫 %s 发现了 %s！"),
           *GetName(), *OtherActor->GetName());

    StartChasing(OtherActor);
}

// =========================================
// 玩家离开视野回调
// =========================================
void APatrolGuard::OnPlayerExitVision(UPrimitiveComponent* OverlappedComp,
                                       AActor* OtherActor,
                                       UPrimitiveComponent* OtherComp,
                                       int32 OtherBodyIndex)
{
    // ----- 1. 安全检查 -----
    if (!OtherActor || OtherActor != CurrentTarget)
    {
        return;
        // 如果不是当前追击的目标离开，不关心
        // 比如另一个AI角色离开视野，不影响追击玩家
    }

    // ----- 2. 不做任何事（让ChaseBehavior处理）-----
    UE_LOG(LogTemp, Log, TEXT("玩家离开视野，警卫记忆 %s，%.1f 秒后回到巡逻"),
           *OtherActor->GetName(), LostTargetMemoryTime);

    // ⚠️ 注意：这里不调用 StopChasing()
    // 因为玩家可能只是短暂离开视野（比如跑到了柱子后面）
    // 我们应该继续追击 LostTargetMemoryTime 秒
    // 这个逻辑在 ChaseBehavior() 的 else 分支中处理
}
```

## 第四步：代码要点详解

### 为什么用ACharacter而不是AActor？

```
AActor
  └── APawn（可被控制器控制，有输入处理）
        └── ACharacter（人形角色）
              ├── 自带 CapsuleComponent（胶囊碰撞体）
              ├── 自带 SkeletalMeshComponent（骨骼模型）
              ├── 自带 CharacterMovementComponent（行走、跳跃、坠落）
              └── 提供 AddMovementInput() 方法

如果用AActor：
  ❌ 需要手动写移动逻辑、碰撞处理、重力
  ❌ 无法利用UE的CharacterMovement系统（极其复杂）

如果用ACharacter：
  ✅ 行走、重力、斜坡、楼梯全部自动处理
  ✅ 改个 MaxWalkSpeed 就切换走/跑速度
  ✅ 未来加跳跃、蹲下只需改几个参数
```

### 视野检测的进阶：射线检测

当前的视野检测只判断距离，实际游戏中需要射线检测判断是否有障碍物挡在中间：

```cpp
// 进阶代码（可选）：在ChaseBehavior中添加射线检测
bool HasLineOfSightTo(AActor* Target)
{
    if (!Target) return false;

    FVector Start = GetActorLocation();
    FVector End = Target->GetActorLocation();

    FHitResult HitResult;
    FCollisionQueryParams QueryParams;
    QueryParams.AddIgnoredActor(this);      // 忽略自己
    QueryParams.AddIgnoredActor(Target);    // 忽略目标
    QueryParams.bTraceComplex = false;      // 使用简化碰撞（更快）

    bool bHit = GetWorld()->LineTraceSingleByChannel(
        HitResult,              // 输出：碰撞结果
        Start,                  // 起点：警卫位置
        End,                    // 终点：玩家位置
        ECC_Visibility,         // 使用可见性通道
        QueryParams             // 查询参数
    );

    // 没碰到任何东西 = 有视线
    return !bHit;
    //  false = 射线没碰到遮挡物 = 能看到
    //  true  = 射线碰到了某物 = 被遮挡
}
```

## 第六步：常见问题排查

### ❌ 警卫不移动

```
可能原因：
1. PatrolPoints 数组为空 → 在蓝图中添加路径点
2. 路径点被删除或无效 → 检查关卡中的PatrolPoint
3. CharacterMovementComponent配置错误 → 检查MaxWalkSpeed是否为0
4. NavMesh没有生成 → ACharacter使用NavMovement需要导航网格
   → 解决方案：关闭导航依赖
   → GetCharacterMovement()->SetMovementMode(MOVE_Walking);
```

### ❌ 警卫不旋转

```
可能原因：
1. bOrientRotationToMovement = true 冲突了
   → 确认构造函数中设为false
2. SmoothLookAt中DirectionToTarget.IsNearlyZero()为true
   → 说明位置重合，检查逻辑
```

### ❌ 警卫不追击玩家

```
可能原因：
1. 玩家没有加 "Player" Tag → 在玩家Actor的Details中添加
2. VisionSphere碰撞设置错误 → 检查是否对ECC_Pawn设为Overlap
3. 玩家和警卫不在同一碰撞通道 → 检查双方的碰撞设置
```

### ❌ 警卫原地转圈或抖动

```
可能原因：
1. 旋转插值速度太快 → 调低TurnSmoothSpeed
2. DeltaTime异常大 → 检查帧率（可能因为断点导致）
3. 路径点位置和警卫位置重合 → 增加ArrivalRadius
```
