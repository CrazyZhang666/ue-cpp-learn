# 4.3 生成与销毁：Actor的出生与死亡

> **目标**：掌握在运行时动态生成（Spawn）和销毁（Destroy）Actor的全部方法，理解SpawnParameters的每个参数，学会延迟生成和对象池等高级技术。

---

## 为什么需要运行时生成/销毁？

在关卡编辑器中拖入Actor是一种"静态放置"，适合场景装饰物。但游戏中的子弹、敌人、特效、道具等需要在**运行时动态出现和消失**。

```
静态放置（编辑器拖入）：          运行时生成（代码中Spawn）：
┌─────────────────────┐         ┌─────────────────────┐
│ • 场景装饰物         │         │ • 子弹/投掷物        │
│ • 固定的门/开关      │         │ • 敌人波次           │
│ • 关卡布局           │         │ • 特效爆炸           │
│ • 不变量             │         │ • 掉落道具           │
└─────────────────────┘         │ • 玩家死亡后的尸体    │
                                └─────────────────────┘
```

---

## SpawnActor — 生成Actor的标准方法

`SpawnActor` 是生成Actor的最常用方法，由 `UWorld` 提供。

### 基本用法（最简形式）

```cpp
void AMyActor::SpawnABullet()
{
    // ===== 步骤1：确保World有效 =====
    UWorld* World = GetWorld();
    if (!World)
    {
        UE_LOG(LogTemp, Error, TEXT("无法获取World！Spawn失败"));
        return;
    }

    // ===== 步骤2：确定生成位置和旋转 =====
    FVector SpawnLocation = GetActorLocation() + GetActorForwardVector() * 100.0f;
    //                                           ↑ 在自身前方100cm处生成

    FRotator SpawnRotation = GetActorRotation();
    // 子弹朝向和发射者一致

    // ===== 步骤3：调用SpawnActor =====
    ABullet* Bullet = World->SpawnActor<ABullet>(
        ABullet::StaticClass(),  // 要生成的类型
        SpawnLocation,           // 位置
        SpawnRotation            // 旋转
    );

    // ===== 步骤4：检查生成是否成功 =====
    if (Bullet)
    {
        UE_LOG(LogTemp, Log, TEXT("子弹生成成功！位置：%s"),
               *SpawnLocation.ToString());
    }
    else
    {
        UE_LOG(LogTemp, Error, TEXT("子弹生成失败！"));
    }
}
```

### 带 SpawnParameters 的完整用法

```cpp
void AMyActor::SpawnWithFullParameters()
{
    UWorld* World = GetWorld();
    if (!World) return;

    // ===== 配置SpawnParameters =====
    FActorSpawnParameters SpawnParams;
    //                       ↑
    // 这个结构体控制Actor怎么被生成
    // 每个参数都有默认值，只需要修改你关心的

    // --- 参数1：Name（Actor的名称）---
    SpawnParams.Name = FName(TEXT("MySpecialBullet"));
    // 如果不设置，引擎自动生成唯一名称如 "Bullet_0", "Bullet_1"...
    // ⚠️ 如果名字已经存在，会在后面加数字后缀

    // --- 参数2：Owner（拥有者）---
    SpawnParams.Owner = this;
    //           ↑
    // Owner不是"父物体"，而是逻辑上的拥有关系
    // 当Owner被销毁时，这个Actor也会被销毁
    // 例如：子弹的Owner是开枪的角色
    //       角色死亡→角色被销毁→子弹也被销毁

    // --- 参数3：Instigator（始作俑者）---
    SpawnParams.Instigator = Cast<APawn>(this);
    //           ↑
    // Instigator通常用于伤害溯源
    // "谁造成的伤害"——用于击杀统计、仇恨系统
    // 例如：玩家开枪→子弹伤害敌人→Instigator是玩家
    //       敌人在日志中显示：被玩家杀死

    // --- 参数4：Template（模板）---
    SpawnParams.Template = nullptr;
    //           ↑
    // 用于从已有Actor复制属性（类似"克隆"）
    // nullptr = 使用类的CDO（Class Default Object，默认配置）

    // --- 参数5：SpawnCollisionHandlingOverride（碰撞处理）---
    SpawnParams.SpawnCollisionHandlingOverride =
        ESpawnActorCollisionHandlingMethod::AdjustIfPossibleButAlwaysSpawn;
    //  四种选项（极其重要！）：

    // 1. Default（使用Actor类的默认设置）
    //    SpawnParams.SpawnCollisionHandlingOverride = ESpawnActorCollisionHandlingMethod::Default;

    // 2. AlwaysSpawn — 不管有没有碰撞，一定生成
    //    ✅ 适用：子弹、特效、UI元素（不需要物理空间的）
    //    ❌ 风险：可能生成在墙里

    // 3. AdjustIfPossibleButAlwaysSpawn — 尝试避开碰撞，不行也生成
    //    ✅ 适用：大多数情况（有备无患）
    //    ❌ 位置可能不是你指定的（被调整了）

    // 4. AdjustIfPossibleButDontSpawnIfColliding — 避开碰撞，不行就不生成
    //    ✅ 适用：需要在空旷位置生成的情况
    //    ❌ 可能生成失败

    // 5. DontSpawnIfColliding — 碰撞就不生成
    //    ✅ 适用：严格位置要求的生成

    // --- 参数6：bDeferConstruction（延迟构造）---
    SpawnParams.bDeferConstruction = false;
    //           ↑
    // true = 先创建对象但不初始化，稍后手动完成构造
    // 用于SpawnActorDeferred（见后面的进阶章节）

    // --- 参数7：bAllowDuringConstructionScript ---
    SpawnParams.bAllowDuringConstructionScript = false;
    //           ↑
    // true = 允许在构造脚本中生成（不常用）

    // --- 参数8：Actor的关卡归属 ---
    SpawnParams.ObjectFlags = RF_Transactional;
    //           ↑
    // RF_Transactional = 允许撤销/重做（编辑器中重要）
    // 还有其他Flag如 RF_Public, RF_Standalone等

    // ===== 执行生成 =====
    ABullet* Bullet = World->SpawnActor<ABullet>(
        ABullet::StaticClass(),
        GetActorLocation() + FVector(100, 0, 50),
        GetActorRotation(),
        SpawnParams  // ← 传入配置好的参数
    );

    if (Bullet)
    {
        // 可以进一步配置生成的Actor
        Bullet->SetLifeSpan(5.0f);  // 5秒后自动销毁
        Bullet->Tags.Add(FName(TEXT("Projectile")));  // 添加标签
    }
}
```

### SpawnParameters 速查表

| 参数 | 类型 | 默认值 | 作用 |
|------|------|--------|------|
| `Name` | FName | NAME_None | 给Actor命名（不设则自动生成） |
| `Owner` | AActor* | nullptr | 逻辑拥有者（Owner销毁则此Actor也被销毁） |
| `Instigator` | APawn* | nullptr | 伤害溯源者（用于击杀统计） |
| `Template` | AActor* | nullptr | 模板Actor（克隆属性用） |
| `SpawnCollisionHandlingOverride` | 枚举 | Default | 碰撞时如何处理 |
| `bDeferConstruction` | bool | false | 是否延迟构造 |
| `bAllowDuringConstructionScript` | bool | false | 是否允许在构造脚本中生成 |
| `ObjectFlags` | EObjectFlags | RF_Transactional | 对象标记（GC相关） |

### 方式1：标准SpawnActor（最常用）

```cpp
ABullet* Bullet = GetWorld()->SpawnActor<ABullet>(
```

### 方式2：使用Transform

```cpp

FTransform SpawnTransform(
    FRotator(0, 90, 0),         // 旋转：面朝Y轴

    SpawnTransform,  // ← 直接传Transform
    SpawnParams

```

### 方式3：SpawnActorDeferred（延迟生成）

```cpp
void AMyActor::SpawnWithCustomInit()
{
    UWorld* World = GetWorld();
    if (!World) return;

    FActorSpawnParameters SpawnParams;
    SpawnParams.bDeferConstruction = true;  // ← 关键！延迟构造
    //                          ↑
    // 设为true后，SpawnActor返回的Actor还没有完成BeginPlay
    // 你可以在BeginPlay之前设置属性

    // 生成"半成品"Actor
    ABullet* Bullet = World->SpawnActor<ABullet>(
        ABullet::StaticClass(),
        SpawnTransform,
        SpawnParams
    );

    if (Bullet)
    {
        // ✅ 在这里设置自定义属性（BeginPlay之前！）
        Bullet->Damage = 50.0f;           // 自定义伤害值
        Bullet->Speed = 2000.0f;          // 自定义速度
        Bullet->OwnerTeam = 1;            // 自定义队伍

        // ⚠️ 以下调用在BeginPlay前不会生效：
        // Bullet->FindComponentByClass<>() ← 组件可能还没完全初始化

        // ===== 手动完成构造 =====
        Bullet->FinishSpawning(SpawnTransform);
        //       ↑
        // 调用FinishSpawning后，BeginPlay才会被调用
        // 此时你设置的属性都已经生效
    }
}
```

### 方式4：从蓝图类生成

```cpp
void AMyActor::SpawnFromBlueprint()
{
    // 假设你有一个蓝图类变量
    UPROPERTY(EditAnywhere, Category = "Spawn")
    TSubclassOf<AActor> ActorToSpawn;  // 编辑器中选择具体蓝图类
    //     ↑

}

// 不用StaticClass()而用TSubclassOf的好处：
// ✅ 策划可以在编辑器中换不同的蓝图实现，不用改C++代码
```

## Actor的销毁

### Destroy() — 最直接的销毁方式

```cpp
void AMyActor::DestroyMyself()
{
    // ✅ 正确：调用Destroy()
    Destroy();
    //    ↑
    // 调用Destroy()后：
    // 1. Actor被标记为"待销毁"（PendingKill）
    // 2. EndPlay() 被调用
    // 3. 所有组件被销毁
    // 4. 下一帧GC回收内存

    // ⚠️ Destroy()不是立即销毁！
    // 在本帧剩余代码中，Actor仍然"存在"但处于待销毁状态
}

// 判断Actor是否已销毁或待销毁
bool IsActorValid(AActor* Actor)
{
    if (!Actor) return false;                    // nullptr
    if (!IsValid(Actor)) return false;           // 已被标记PendingKill
    if (Actor->IsPendingKill()) return false;    // 同上，更具体
    if (Actor->IsActorBeingDestroyed()) return false; // 正在销毁中
    return true;  // Actor有效
}
```

### SetLifeSpan() — 定时自动销毁

```cpp
void AMyActor::SpawnExpiringBullet()
{
    ABullet* Bullet = GetWorld()->SpawnActor<ABullet>(
        ABullet::StaticClass(),
        GetActorLocation(),
        GetActorRotation()
    );

    if (Bullet)
    {
        // 方式1：设置生命周期（秒）
        Bullet->SetLifeSpan(5.0f);
        //     ↑ 5秒后自动调用Destroy()
        // 适合：子弹、临时特效、限时道具

        // 方式2：获取剩余生命时间
        float RemainingTime = Bullet->GetLifeSpan();
        UE_LOG(LogTemp, Log, TEXT("子弹将在 %.1f 秒后销毁"), RemainingTime);

        // 方式3：取消自动销毁
        // Bullet->SetLifeSpan(0.0f);  // 设为0 = 取消自动销毁
    }
}

// 或者直接在子弹类的BeginPlay中设置
void ABullet::BeginPlay()
{
    Super::BeginPlay();

    // 所有子弹默认10秒后自动销毁（防止飞出地图永不消失）
    SetLifeSpan(10.0f);
}
```

### 使用Timer延迟销毁

```cpp
void AMyActor::DestroyAfterDelay()
{
    // 方式1：使用FTimerHandle（更灵活）
    FTimerHandle DestroyTimer;

    GetWorld()->GetTimerManager().SetTimer(
        DestroyTimer,           // Timer句柄（可以用来取消）
        [this]()                // Lambda回调
        {
            if (IsValid(this))
            {
                UE_LOG(LogTemp, Log, TEXT("时间到，销毁Actor"));
                Destroy();
            }
        },
        3.0f,                   // 延迟3秒
        false                   // false = 只执行一次，true = 循环
    );

    // 如果中途想取消延迟销毁：
    // GetWorld()->GetTimerManager().ClearTimer(DestroyTimer);
}

// 方式2：使用SetLifeSpan（更简单，推荐大多数情况）
void AMyActor::DestroyAfterDelaySimple()
{
    SetLifeSpan(3.0f);  // 3秒后自动销毁，一行搞定
}
```

## 对象池（Object Pool）

### 为什么需要对象池？

在游戏中频繁生成和销毁Actor（如子弹、特效）非常消耗性能。每Spawn一次都需要分配内存，每Destroy一次都要等GC回收。

```
频繁 Spawn + Destroy 的开销：
┌──────────────────────────────────────────────┐
│ 生成 1000 颗子弹：                            │
│   分配内存：1000 次                           │
│   GC回收：  1000 次                           │
│   总耗时：可能每帧 > 20ms（导致掉帧）          │
│                                              │
│ 对象池（1000颗子弹）：                        │
│   初始创建：1000 次（仅一次）                  │
│   "生成"：  从池里取一个（几乎0开销）          │
│   "销毁"：  放回池里（几乎0开销）              │
│   总耗时：每帧 < 1ms                           │
└──────────────────────────────────────────────┘
```

### 简单的对象池实现

```cpp
// ========== ObjectPool.h ==========
#pragma once

#include "CoreMinimal.h"
#include "GameFramework/Actor.h"
#include "ObjectPool.generated.h"

/**
 * 简单的对象池管理器
 *
 * 用法：
 * 1. 先调用 InitializePool<ABullet>(100) 预创建100颗子弹
 * 2. 需要子弹时调用 GetFromPool<ABullet>()
 * 3. 子弹用完调用 ReturnToPool(Bullet)
 */
UCLASS()
class MYGAME_API AObjectPool : public AActor
{
    GENERATED_BODY()

public:
    AObjectPool();

protected:
    virtual void BeginPlay() override;

public:
    /**
     * 初始化某种类型的对象池
     * @param Size - 池子大小（预创建多少个）
     */
    template<typename T>
    void InitializePool(int32 Size)
    {
        // 用模板类型作为Key（每个类型的池子独立）
        FString PoolKey = T::StaticClass()->GetName();

        // 确保该Key对应的数组存在
        if (!PoolMap.Contains(PoolKey))
        {
            PoolMap.Add(PoolKey, FPoolArray());
        }

        FPoolArray& Pool = PoolMap[PoolKey];

        // 预创建Size个Actor，初始设为隐藏+停用
        for (int32 i = 0; i < Size; i++)
        {
            // 生成Actor
            FActorSpawnParameters SpawnParams;
            SpawnParams.SpawnCollisionHandlingOverride =
                ESpawnActorCollisionHandlingMethod::AlwaysSpawn;

            T* NewObj = GetWorld()->SpawnActor<T>(
                T::StaticClass(),
                FVector::ZeroVector,    // 初始位置（池子里无所谓）
                FRotator::ZeroRotator,
                SpawnParams
            );

            if (NewObj)
            {
                // 标为不可见，放入池中
                NewObj->SetActorHiddenInGame(true);     // 隐藏
                NewObj->SetActorEnableCollision(false);  // 关闭碰撞
                NewObj->SetActorTickEnabled(false);      // 关闭Tick

                Pool.Add(NewObj);  // 加入池子
            }
        }

        UE_LOG(LogTemp, Log, TEXT("对象池 [%s] 初始化完成，预创建了 %d 个对象"),
               *PoolKey, Size);
    }

    /**
     * 从池子中获取一个可用的Actor
     * 如果池子空了，动态创建新的
     */
    template<typename T>
    T* GetFromPool()
    {
        FString PoolKey = T::StaticClass()->GetName();

        // 查找对应池子
        if (FPoolArray* Pool = PoolMap.Find(PoolKey))
        {
            // 遍历找可用的（被隐藏的就是可用的）
            for (AActor* Actor : *Pool)
            {
                if (Actor && Actor->IsHidden())  // 隐藏 = 在池子里待用
                {
                    // "激活"它
                    Actor->SetActorHiddenInGame(false);      // 显示
                    Actor->SetActorEnableCollision(true);    // 开启碰撞
                    Actor->SetActorTickEnabled(true);        // 开启Tick

                    // 通过自定义接口重置状态（让子类实现）
                    if (IPoolable* PoolableObj = Cast<IPoolable>(Actor))
                    {
                        PoolableObj->OnSpawnFromPool();
                    }

                    return Cast<T>(Actor);
                }
            }

            // 池子空了！动态创建一个新的
            UE_LOG(LogTemp, Warning, TEXT("对象池 [%s] 已空，正在动态扩展"), *PoolKey);

            T* NewObj = GetWorld()->SpawnActor<T>(T::StaticClass());
            if (NewObj)
            {
                Pool->Add(NewObj);
            }
            return NewObj;
        }

        UE_LOG(LogTemp, Error, TEXT("对象池 [%s] 未初始化！"), *PoolKey);
        return nullptr;
    }

    /**
     * 将Actor归还到池子
     */
    void ReturnToPool(AActor* Actor)
    {
        if (!IsValid(Actor)) return;

        // "停用"并放回池子
        Actor->SetActorHiddenInGame(true);
        Actor->SetActorEnableCollision(false);
        Actor->SetActorTickEnabled(false);

        // 移动到远离场景的位置（避免和任何东西碰撞）
        Actor->SetActorLocation(FVector(0, 0, -100000));
    }

private:
    // 用TArray<AActor*>当池子
    using FPoolArray = TArray<AActor*>;

    // Key=类名, Value=该类的池子
    TMap<FString, FPoolArray> PoolMap;
};

/**
 * 可入池对象的接口
 * 如果你的Actor需要池化，实现这个接口
 */
UINTERFACE()
class UPoolable : public UInterface
{
    GENERATED_BODY()
};

class IPoolable : public IInterface
{
    GENERATED_BODY()

public:
    /** 当对象从池中被取出来时调用（相当于BeginPlay，但每次取都调） */
    virtual void OnSpawnFromPool() = 0;

    /** 当对象被放回池子时调用 */
    virtual void OnReturnToPool() = 0;
};
```

### 对象池的使用示例

```cpp
// ========== PooledBullet.h ==========
#pragma once

#include "CoreMinimal.h"
#include "GameFramework/Actor.h"
#include "ObjectPool.h"  // 包含 IPoolable 接口
#include "PooledBullet.generated.h"

UCLASS()
class MYGAME_API APooledBullet : public AActor, public IPoolable
//                               ↑              ↑
//                           AActor   IPoolable接口（可入池）
{
    GENERATED_BODY()

public:
    APooledBullet();

    virtual void OnSpawnFromPool() override;
    virtual void OnReturnToPool() override;

    // 发射子弹（设置初速度和位置）
    void Fire(const FVector& StartLocation, const FVector& Direction, float Speed);

protected:
    virtual void Tick(float DeltaTime) override;

private:
    FVector Velocity;  // 飞行速度（方向*速率）
    float LifeTime;    // 已飞行时间
    float MaxLifeTime = 5.0f;  // 最大飞行时间
};

// ========== PooledBullet.cpp ==========
#include "PooledBullet.h"

APooledBullet::APooledBullet()
{
    PrimaryActorTick.bCanEverTick = true;

    // 创建简单的RootComponent
    USceneComponent* Root = CreateDefaultSubobject<USceneComponent>(TEXT("Root"));
    RootComponent = Root;
}

void APooledBullet::OnSpawnFromPool()
{
    // 从池子取出时重置状态
    LifeTime = 0.0f;
    Velocity = FVector::ZeroVector;

    UE_LOG(LogTemp, Log, TEXT("子弹从池子中取出，准备发射"));
}

void APooledBullet::OnReturnToPool()
{
    // 放回池子前的清理
    Velocity = FVector::ZeroVector;

    UE_LOG(LogTemp, Log, TEXT("子弹放回池子"));
}

void APooledBullet::Fire(const FVector& StartLocation, const FVector& Direction, float Speed)
{
    // 设置初始位置
    SetActorLocation(StartLocation);

    // 计算速度
    Velocity = Direction.GetSafeNormal() * Speed;
    //                     ↑ 保证Direction长度为1，再乘以速率
    LifeTime = 0.0f;
}

void APooledBullet::Tick(float DeltaTime)
{
    Super::Tick(DeltaTime);

    // 移动子弹
    FVector NewLocation = GetActorLocation() + Velocity * DeltaTime;
    SetActorLocation(NewLocation);

    // 超时自动归还池子
    LifeTime += DeltaTime;
    if (LifeTime >= MaxLifeTime)
    {
        // 找到池子管理器并归还
        // 实际项目中用单例或GameInstance管理池子管理器
        // AObjectPool* Pool = GetObjectPool();
        // Pool->ReturnToPool(this);
    }
}
```

## 关卡流式加载与Actor

在大型开放世界游戏中，关卡是分块流式加载的。理解Actor在其中的行为很重要。

```cpp
// ===== 流式关卡中Actor的注意事项 =====

void AMyActor::BeginPlay()
{
    Super::BeginPlay();

    // 获取当前Actor所在的关卡
    ULevel* MyLevel = GetLevel();
    if (MyLevel)
    {
        FString LevelName = MyLevel->GetOutermost()->GetName();
        UE_LOG(LogTemp, Log, TEXT("我在关卡：%s"), *LevelName);

        // 检查关卡是否可见（已加载并显示）
        bool bIsVisible = MyLevel->bIsVisible;
        UE_LOG(LogTemp, Log, TEXT("关卡可见：%s"), bIsVisible ? TEXT("是") : TEXT("否"));
    }

    // ⚠️ 流式关卡卸载时：
    // - 该关卡中的所有Actor都会被销毁
    // - EndPlay() 会被调用，EndPlayReason = RemovedFromWorld
    // - 如果Actor需要跨关卡存在（如玩家），需要特殊处理

    // 标记Actor为持久存在（不随关卡卸载）
    // SetActorHiddenInGame(false); ← 这不阻止卸载
    // 需要：Actor放在持久关卡（PersistentLevel）中
}
```

## 实际案例：完整的子弹系统

将生成、飞行、碰撞检测、自动销毁整合到一起。

```cpp
// ========== Bullet.h ==========
#pragma once

#include "CoreMinimal.h"
#include "GameFramework/Actor.h"
#include "Bullet.generated.h"

class USphereComponent;
class UProjectileMovementComponent;

UCLASS()
class MYGAME_API ABullet : public AActor
{
    GENERATED_BODY()

public:
    ABullet();

protected:
    virtual void BeginPlay() override;

public:
    // ===== 组件 =====
    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "Components")
    USphereComponent* CollisionSphere;  // 碰撞球（检测命中）

    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "Components")
    UStaticMeshComponent* BulletMesh;  // 子弹模型

    UPROPERTY(VisibleAnywhere, BlueprintReadOnly, Category = "Components")
    UProjectileMovementComponent* ProjectileMovement;  // 弹道移动组件

    // ===== 属性 =====
    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Bullet")
    float Damage = 25.0f;  // 伤害值

    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Bullet")
    float BulletSpeed = 3000.0f;  // 子弹速度（cm/s = 30m/s）

    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Bullet")
    float MaxDistance = 10000.0f;  // 最大飞行距离（100米）

    UPROPERTY(EditAnywhere, BlueprintReadWrite, Category = "Bullet")
    bool bUseGravity = false;  // 是否有重力（子弹：false，榴弹：true）

    // 初始化子弹（由发射者调用）
    void Initialize(const FVector& Direction, AActor* Shooter);

    // 碰撞回调
    UFUNCTION()
    void OnBulletHit(UPrimitiveComponent* HitComp, AActor* OtherActor,
                     UPrimitiveComponent* OtherComp, FVector NormalImpulse,
                     const FHitResult& Hit);

private:
    FVector StartLocation;  // 发射位置（用于计算飞行距离）
    AActor* OwnerShooter;   // 发射者（用于防止自伤）
};

// ========== Bullet.cpp ==========
#include "Bullet.h"
#include "Components/SphereComponent.h"
#include "Components/StaticMeshComponent.h"
#include "GameFramework/ProjectileMovementComponent.h"

ABullet::ABullet()
{
    PrimaryActorTick.bCanEverTick = false;  // 子弹用ProjectileMovement，不需要手动Tick

    // ===== 1. 创建碰撞球 =====
    CollisionSphere = CreateDefaultSubobject<USphereComponent>(TEXT("CollisionSphere"));
    RootComponent = CollisionSphere;  // ← 直接用碰撞球做RootComponent！
    CollisionSphere->SetSphereRadius(10.0f);  // 1cm半径的小球

    // 碰撞设置：阻挡所有，生成Overlap事件
    CollisionSphere->SetCollisionEnabled(ECollisionEnabled::QueryAndPhysics);
    CollisionSphere->SetCollisionObjectType(ECC_WorldDynamic);
    CollisionSphere->SetCollisionResponseToAllChannels(ECR_Block);
    // 子弹阻挡所有东西（碰到就停）

    // ===== 2. 创建子弹模型 =====
    BulletMesh = CreateDefaultSubobject<UStaticMeshComponent>(TEXT("BulletMesh"));
    BulletMesh->SetupAttachment(CollisionSphere);  // 挂在碰撞球下
    BulletMesh->SetCollisionEnabled(ECollisionEnabled::NoCollision);
    // 模型不参与碰撞（碰撞由CollisionSphere处理）
    BulletMesh->SetRelativeScale3D(FVector(0.1f, 0.1f, 0.5f));
    // 缩放成子弹形状（细长）

    // ===== 3. 创建弹道移动组件 =====
    ProjectileMovement = CreateDefaultSubobject<UProjectileMovementComponent>(
        TEXT("ProjectileMovement")
    );
    // 不需要SetupAttachment！MovementComponent是纯逻辑组件

    // 配置弹道移动
    ProjectileMovement->InitialSpeed = BulletSpeed;    // 初速
    ProjectileMovement->MaxSpeed = BulletSpeed;        // 最大速度
    ProjectileMovement->bRotationFollowsVelocity = true; // 旋转跟随速度方向
    ProjectileMovement->bShouldBounce = false;         // 不反弹（碰到就停）
    ProjectileMovement->ProjectileGravityScale = 0.0f;  // 无重力

    // ===== 4. 初始隐藏 =====
    // 子弹生成后先隐藏，等Fire调用Initialize再显示
    // SetActorHiddenInGame(true);

    // ===== 5. 默认自动销毁 =====
    InitialLifeSpan = 10.0f;
    //  ↑
    // InitialLifeSpan 在构造函数中设置，等价于BeginPlay中调用SetLifeSpan
    // 子弹即使没碰到任何东西，10秒后也会自动销毁
}

void ABullet::BeginPlay()
{
    Super::BeginPlay();

    // 记录初始位置（用于计算飞行距离）
    StartLocation = GetActorLocation();

    // 绑定碰撞回调
    CollisionSphere->OnComponentHit.AddDynamic(this, &ABullet::OnBulletHit);
}

void ABullet::Initialize(const FVector& Direction, AActor* Shooter)
{
    // 记录发射者
    OwnerShooter = Shooter;

    // 设置子弹速度
    ProjectileMovement->Velocity = Direction.GetSafeNormal() * BulletSpeed;
    //                              ↑
    // 直接设置Velocity会立即改变移动方向

    // 设置重力
    ProjectileMovement->ProjectileGravityScale = bUseGravity ? 1.0f : 0.0f;

    // 让子弹忽略发射者的碰撞（防止自伤）
    if (OwnerShooter)
    {
        CollisionSphere->IgnoreActorWhenMoving(OwnerShooter, true);
        //                                     ↑
        // 移动时忽略发射者，这样子弹不会刚发出就打到自己
    }

    UE_LOG(LogTemp, Log, TEXT("子弹发射！速度：%.0f cm/s，伤害：%.1f"),
           BulletSpeed, Damage);
}

void ABullet::OnBulletHit(UPrimitiveComponent* HitComp, AActor* OtherActor,
                           UPrimitiveComponent* OtherComp, FVector NormalImpulse,
                           const FHitResult& Hit)
{
    // ===== 1. 安全检查 =====
    if (!OtherActor || OtherActor == this || OtherActor == OwnerShooter)
    {
        return;  // 忽略自碰和碰发射者
    }

    // ===== 2. 日志 =====
    UE_LOG(LogTemp, Log, TEXT("子弹命中：%s，飞行距离：%.0f cm"),
           *OtherActor->GetName(),
           FVector::Dist(StartLocation, GetActorLocation()));

    // ===== 3. 造成伤害 =====
    // 对命中的Actor应用伤害（需要该Actor有处理伤害的逻辑）
    FPointDamageEvent DamageEvent(Damage, Hit, NormalImpulse, nullptr);
    //                           伤害值 命中点 法线方向  伤害类型
    OtherActor->TakeDamage(Damage, DamageEvent, nullptr, OwnerShooter);
    //          ↑                                                     ↑
    //    受到伤害的Actor                                    Instigator（谁造成的伤害）

    // ===== 4. 在命中点生成命中特效 =====
    // 实际项目中会有一个命中特效管理器
    // UGameplayStatics::SpawnEmitterAtLocation(GetWorld(), HitEffect, Hit.Location);

    // ===== 5. 销毁子弹 =====
    Destroy();
}
```

```cpp
void AMyWeapon::FireBullet()
{
    UWorld* World = GetWorld();
    if (!World) return;

    // ----- 计算发射位置和方向 -----
    // 枪口位置（假设武器Actor的前方100cm处）

        ABullet::StaticClass(),
        MuzzleLocation,
        FireDirection.Rotation(),  // 从方向向量转为旋转
        SpawnParams

    // ----- 初始化子弹 -----
    if (Bullet)

        Bullet->Damage = 25.0f;          // 本次射击的伤害
        Bullet->Initialize(FireDirection, SpawnParams.Instigator);
    }
```
