## 案例概述

### 我们要做什么

实现一个功能完整的"多人对战射击"核心网络系统，包含：

```
玩家功能：
  ✅ 开火射击（Server RPC + 服务器验证）
  ✅ 血量系统（Replicated + OnRep）
  ✅ 护甲系统（ReplicatedUsing）
  ✅ 弹药管理（ReplicatedUsing + COND_OwnerOnly）
  ✅ 死亡/重生（服务器权威）
  ✅ 枪口火焰特效（NetMulticast RPC）
  ✅ 伤害数字显示（Client RPC）

服务器功能：
  ✅ 玩家加入/离开处理（PostLogin / Logout）
  ✅ 击杀统计（服务器权威）
  ✅ 玩家列表管理

子弹系统：
  ✅ 服务器端碰撞检测（服务器权威）
  ✅ 位置同步（Replicated）
```

## ANetworkCharacter — 网络角色类

### 头文件完整实现

```cpp
// ============================================
// NetworkCharacter.h
// 多人对战射击游戏的核心角色类
// ============================================
#pragma once

#include "CoreMinimal.h"
#include "GameFramework/Character.h"
#include "NetworkCharacter.generated.h"

// 前向声明：引用的子弹类
class ANetworkProjectile;

UCLASS()
class MYGAME_API ANetworkCharacter : public ACharacter
{
    GENERATED_BODY()

public:
    // ============================================
    // 构造函数
    // ============================================
    ANetworkCharacter();

    // ============================================
    // 同步属性 — 血量
    // ============================================
    // ReplicatedUsing：当客户端收到服务器同步的新值时
    // 自动调用 OnRep_Health() 回调函数
    UPROPERTY(ReplicatedUsing = OnRep_Health)
    float Health;

    // 最大血量（这是一个常量配置，不需要同步——所有端都一样）
    UPROPERTY(EditDefaultsOnly, Category = "Combat")
    float MaxHealth;

    // ============================================
    // 同步属性 — 护甲
    // ============================================
    UPROPERTY(ReplicatedUsing = OnRep_Armor)
    float Armor;

    UPROPERTY(EditDefaultsOnly, Category = "Combat")
    float MaxArmor;

    // ============================================
    // 同步属性 — 弹药
    // ============================================
    // 弹药只对有拥有者客户端可见（COND_OwnerOnly）
    // 因为别人不需要知道你还剩多少子弹
    UPROPERTY(ReplicatedUsing = OnRep_Ammo)
    int32 Ammo;

    UPROPERTY(EditDefaultsOnly, Category = "Combat")
    int32 MaxAmmo;

    // ============================================
    // 同步属性 — 存活状态
    // ============================================
    UPROPERTY(Replicated)
    bool bIsAlive;

    // ============================================
    // 同步属性 — 击杀数
    // ============================================
    UPROPERTY(Replicated)
    int32 KillCount;

    // ============================================
    // 同步属性 — 死亡数
    // ============================================
    UPROPERTY(Replicated)
    int32 DeathCount;

    // ============================================
    // OnRep 回调函数声明
    // ============================================
    // 这些函数在客户端收到属性更新时自动被调用
    UFUNCTION()
    void OnRep_Health();

    UFUNCTION()
    void OnRep_Armor();

    UFUNCTION()
    void OnRep_Ammo();

    // ============================================
    // Server RPC — 客户端请求，服务器执行
    // ============================================

    // 客户端请求开火
    // 参数 AimDirection：客户端的瞄准方向
    UFUNCTION(Server, Reliable, WithValidation)
    void ServerFireWeapon(FVector AimDirection);

    // 客户端请求换弹
    UFUNCTION(Server, Reliable, WithValidation)
    void ServerReload();

    // 客户端请求重生（通常在死亡后调用）
    UFUNCTION(Server, Reliable, WithValidation)
    void ServerRequestRespawn();

    // ============================================
    // Client RPC — 服务器调用，拥有者客户端执行
    // ============================================

    // 对被击中的玩家显示伤害数字
    // 只有被击中的那一个客户端能看到这个数字
    UFUNCTION(Client, Reliable)
    void ClientShowDamageNumber(float DamageAmount, bool bIsHeadshot);

    // 通知客户端更新弹药（用于纠正本地预测的弹药数）
    UFUNCTION(Client, Reliable)
    void ClientCorrectAmmo(int32 CorrectAmmo);

    // 显示击杀提示（给击杀者看的）
    UFUNCTION(Client, Reliable)
    void ClientShowKillConfirmation(const FString& VictimName);

    // ============================================
    // NetMulticast RPC — 服务器调用，所有客户端执行
    // ============================================

    // 所有客户端看到枪口火焰特效
    UFUNCTION(NetMulticast, Unreliable)  // Unreliable：特效丢了也无所谓
    void MulticastPlayMuzzleFlash();

    // 所有客户端看到命中特效（火花）
    UFUNCTION(NetMulticast, Reliable)
    void MulticastPlayHitEffect(FVector HitLocation, bool bIsHeadshot);

    // 所有客户端看到死亡动画
    UFUNCTION(NetMulticast, Reliable)
    void MulticastPlayDeathAnimation();

    // 所有客户端看到重生效果
    UFUNCTION(NetMulticast, Reliable)
    void MulticastPlayRespawnEffect();

    // ============================================
    // 网络属性注册
    // ============================================
    virtual void GetLifetimeReplicatedProps(
        TArray<FLifetimeProperty>& OutLifetimeProps
    ) const override;

    // ============================================
    // 服务器端逻辑函数（非RPC，只在服务器调用）
    // ============================================
    void ServerApplyDamage(float Damage, ANetworkCharacter* DamageInstigator, bool bIsHeadshot);
    void ServerHandleDeath(ANetworkCharacter* Killer);
    void ServerRespawn();

    // ============================================
    // 输入处理（本地客户端）
    // ============================================
    void OnFirePressed();
    void OnReloadPressed();

protected:
    virtual void BeginPlay() override;

private:
    // ============================================
    // 武器配置
    // ============================================
    // 每发子弹的伤害值
    UPROPERTY(EditDefaultsOnly, Category = "Weapon", meta = (AllowPrivateAccess = "true"))
    float WeaponDamage;

    // 暴击伤害倍数（用于爆头等场景）
    UPROPERTY(EditDefaultsOnly, Category = "Weapon", meta = (AllowPrivateAccess = "true"))
    float HeadshotMultiplier;

    // 射击冷却时间（秒）
    UPROPERTY(EditDefaultsOnly, Category = "Weapon", meta = (AllowPrivateAccess = "true"))
    float FireRate;

    // 最大射击距离（单位：cm，1单位=1cm，所以这是100米）
    UPROPERTY(EditDefaultsOnly, Category = "Weapon", meta = (AllowPrivateAccess = "true"))
    float MaxFireRange;

    // 子弹类（用于生成投射物）
    UPROPERTY(EditDefaultsOnly, Category = "Weapon", meta = (AllowPrivateAccess = "true"))
    TSubclassOf<ANetworkProjectile> ProjectileClass;

    // 枪口火焰粒子特效
    UPROPERTY(EditDefaultsOnly, Category = "Effects", meta = (AllowPrivateAccess = "true"))
    UParticleSystem* MuzzleFlashEffect;

    // 命中火花粒子特效
    UPROPERTY(EditDefaultsOnly, Category = "Effects", meta = (AllowPrivateAccess = "true"))
    UParticleSystem* HitImpactEffect;

    // 开火音效
    UPROPERTY(EditDefaultsOnly, Category = "Effects", meta = (AllowPrivateAccess = "true"))
    USoundBase* FireSound;

    // ============================================
    // 内部状态
    // ============================================
    // 是否可以开火（冷却控制）
    bool bCanFire;

    // 射击冷却计时器句柄
    FTimerHandle FireRateTimerHandle;

    // 上次开火时间
    float LastFireTime;

    // 本地预测的弹药数（用于开火时立即减少UI显示）
    int32 LocalPredictedAmmo;
};
```

### 实现文件完整代码

```cpp
// ============================================
// NetworkCharacter.cpp
// 多人对战射击游戏核心角色实现
// ============================================
#include "NetworkCharacter.h"
#include "NetworkProjectile.h"
#include "NetworkGameMode.h"
#include "NetworkPlayerController.h"
#include "Net/UnrealNetwork.h"           // ← 网络复制支持
#include "Engine/Engine.h"               // ← UE_LOG
#include "GameFramework/PlayerController.h"
#include "Kismet/GameplayStatics.h"      // ← 静态函数库
#include "Camera/CameraComponent.h"      // ← 摄像机
#include "Components/CapsuleComponent.h" // ← 碰撞胶囊体

// ============================================
// 构造函数
// ============================================
ANetworkCharacter::ANetworkCharacter()
{
    // ===== 设置默认值 =====
    // 这些默认值可以在蓝图中被 EditDefaultsOnly 覆盖

    // 战斗属性
    MaxHealth = 100.0f;
    Health = MaxHealth;          // 初始满血
    MaxArmor = 50.0f;
    Armor = 0.0f;                // 初始无护甲
    MaxAmmo = 300;
    Ammo = 30;                   // 初始30发子弹
    bIsAlive = true;             // 初始存活
    KillCount = 0;
    DeathCount = 0;

    // 武器配置
    WeaponDamage = 25.0f;        // 每发25点伤害
    HeadshotMultiplier = 2.0f;   // 爆头2倍伤害
    FireRate = 0.1f;             // 每0.1秒可以开一枪（10发/秒）
    MaxFireRange = 100000.0f;    // 1000米（100000cm）

    // 内部状态
    bCanFire = true;             // 初始可以开火
    LastFireTime = 0.0f;
    LocalPredictedAmmo = Ammo;

    // ===== 网络设置 =====
    // ★★★ 重要：必须设置 bReplicates = true ★★★
    // 否则这个 Actor 不会通过网络同步
    bReplicates = true;

    // 移动组件自动处理移动同步
    // CharacterMovementComponent 的 bReplicateMovement 默认为 true

    // 设置网络更新频率
    // 玩家角色需要较高频率以保证流畅的移动同步
    NetUpdateFrequency = 30.0f;    // 每秒30次（大约每0.03秒更新一次）

    // 设置网络优先级
    // 玩家角色是最高优先级的网络对象
    NetPriority = 3.0f;

    // 最小网络更新频率
    // 即使角色不动，也要保证每0.5秒至少同步一次（"心跳"）
    MinNetUpdateFrequency = 2.0f;
}

// ============================================
// BeginPlay — 游戏开始时调用
// ============================================
void ANetworkCharacter::BeginPlay()
{
    Super::BeginPlay();

    // ===== 初始化健康状态 =====
    // 确保所有端有正确的初始值
    Health = MaxHealth;
    Armor = MaxArmor;
    Ammo = MaxAmmo;
    bIsAlive = true;
    bCanFire = true;
    LocalPredictedAmmo = Ammo;

    // ===== 只在服务器上执行的初始化 =====
    if (HasAuthority())
    {
        // 服务器端特定的初始化逻辑
        // 例如：注册到 GameMode 的玩家列表中等等
        UE_LOG(LogTemp, Log, TEXT("服务器：角色 %s 已初始化"), *GetName());
    }

    // ===== 只在本地客户端执行的初始化 =====
    if (IsLocallyControlled())
    {
        // 本地玩家特定的初始化
        // 例如：设置摄像机的后处理效果、初始化第一人称UI等
        UE_LOG(LogTemp, Log, TEXT("客户端：本地角色 %s 已初始化"), *GetName());
    }
}

// ============================================
// GetLifetimeReplicatedProps — 注册需要同步的属性
// ============================================
void ANetworkCharacter::GetLifetimeReplicatedProps(
    TArray<FLifetimeProperty>& OutLifetimeProps
) const
{
    // ★ 第一步：调用父类的注册函数（必须！）
    // Character 父类有自己的同步属性（如位置、旋转等）
    Super::GetLifetimeReplicatedProps(OutLifetimeProps);

    // ★ 第二步：注册本类中的同步属性

    // 无条件同步：所有客户端都需要知道这些信息
    DOREPLIFETIME(ANetworkCharacter, Health);       // 血量—显示血条需要
    DOREPLIFETIME(ANetworkCharacter, Armor);         // 护甲—显示护甲条需要
    DOREPLIFETIME(ANetworkCharacter, bIsAlive);      // 存活状态—判断是否显示角色
    DOREPLIFETIME(ANetworkCharacter, KillCount);     // 击杀数—计分板需要
    DOREPLIFETIME(ANetworkCharacter, DeathCount);    // 死亡数—计分板需要

    // 条件同步：弹药只同步给拥有者客户端（别人不需要知道你的弹药量）
    DOREPLIFETIME_CONDITION(ANetworkCharacter, Ammo, COND_OwnerOnly);
}

// ============================================
// 输入处理
// ============================================
void ANetworkCharacter::OnFirePressed()
{
    // ===== 第一步：基本检查 =====
    // 必须是我在控制的角色（不是别人的角色）
    if (!IsLocallyControlled())
    {
        return;  // 这不是我的角色，直接返回
    }

    // ===== 第二步：本地预测 =====
    // 不等服务器确认，直接在本地做可以做的事情：

    // 预测性弹药减少（不等服务器确认就先更新UI）
    LocalPredictedAmmo = FMath::Max(0, LocalPredictedAmmo - 1);
    // 伪代码：UpdateAmmoUI(LocalPredictedAmmo);
    // ↑ 这样按开火键后UI立即响应，不会有延迟感

    // 预测性枪口火焰（不等多播RPC返回）
    // 伪代码：PlayLocalMuzzleFlash();
    // ↑ 本地立即显示枪口火焰，服务端确认后可以通过多播RPC再次播放
    //   （重复播放枪口火焰问题不大，视觉上不明显）

    // ===== 第三步：发送 Server RPC =====
    // 获取当前瞄准方向（根据摄像机的朝向）
    FVector AimDirection = GetControlRotation().Vector();

    // 发送到服务器
    // ★★★ 注意：ServerFireWeapon 是 Server RPC ★★★
    // 客户端调用这个函数 → 实际上是通过网络发送给服务器
    // 客户端本身不会执行 _Implementation 中的逻辑
    ServerFireWeapon(AimDirection);
}

void ANetworkCharacter::OnReloadPressed()
{
    // 必须是我控制的角色
    if (!IsLocallyControlled())
    {
        return;
    }

    // 发送换弹请求给服务器
    ServerReload();
}

// ============================================
// Server RPC 实现 — ServerFireWeapon
// ============================================

// Validate 函数：参数验证
bool ANetworkCharacter::ServerFireWeapon_Validate(FVector AimDirection)
{
    // ===== 轻量级数据验证 =====
    // 这里只验证数据格式是否合法，不验证游戏规则

    // 验证1：方向向量不能包含 NaN（不是一个数字）
    if (AimDirection.ContainsNaN())
    {
        UE_LOG(LogTemp, Warning, TEXT("SUSPICIOUS：开火方向包含NaN！"));
        return false;  // 拒绝这个RPC调用
    }

    // 验证2：方向向量长度应该接近1（标准化后的向量）
    float Length = AimDirection.Size();
    if (Length < 0.5f || Length > 1.5f)
    {
        UE_LOG(LogTemp, Warning, TEXT("SUSPICIOUS：开火方向长度异常 %.2f"), Length);
        return false;
    }

    // 验证通过
    return true;
}

// Implementation 函数：服务器端实际逻辑
void ANetworkCharacter::ServerFireWeapon_Implementation(FVector AimDirection)
{
    // ============================================
    // 重要：这个函数只在服务器上执行！
    // 尽管客户端调用了 ServerFireWeapon()，
    // 但实际逻辑是在服务器端运行的。
    // ============================================

    // ===== 游戏规则验证 =====
    // 这些验证在 _Validate 通过后执行
    // 用于检查游戏规则的合法性

    // 验证1：角色是否存活？
    if (!bIsAlive)
    {
        UE_LOG(LogTemp, Warning, TEXT("开火被拒绝：角色 %s 已死亡"), *GetName());
        return;
    }

    // 验证2：是否有弹药？
    if (Ammo <= 0)
    {
        UE_LOG(LogTemp, Warning, TEXT("开火被拒绝：角色 %s 没有弹药"), *GetName());
        // 注意：这里可以通知客户端纠正它的预测弹药数
        // 因为客户端可能显示了错误的弹药数
        ClientCorrectAmmo(Ammo);  // 告诉客户端真正的弹药数（0）
        return;
    }

    // 验证3：冷却时间检查（防止射速外挂）
    float CurrentTime = GetWorld()->GetTimeSeconds();
    float TimeSinceLastFire = CurrentTime - LastFireTime;
    if (TimeSinceLastFire < FireRate)
    {
        UE_LOG(LogTemp, Warning, TEXT("开火被拒绝：冷却中（%.3f < %.3f）"),
            TimeSinceLastFire, FireRate);
        return;  // 射得太快，拒绝！
    }

    // 验证4：是否正在换弹？
    // if (bIsReloading) { return; }

    // ===== 全部验证通过！执行开火逻辑 =====

    // 1. 消耗弹药（在服务器端）
    Ammo--;
    LastFireTime = CurrentTime;

    UE_LOG(LogTemp, Log, TEXT("服务器：%s 开火！剩余弹药：%d"), *GetName(), Ammo);

    // 2. 通知所有客户端播放枪口火焰（NetMulticast RPC）
    MulticastPlayMuzzleFlash();

    // 3. 执行服务器端射线检测（LineTrace）
    FVector StartLocation = GetActorLocation();
    // 从角色的眼睛位置发出射线（更准确）
    FVector EyeLocation;
    FRotator EyeRotation;
    GetActorEyesViewPoint(EyeLocation, EyeRotation);

    // 计算射线的终点
    FVector EndLocation = EyeLocation + (AimDirection * MaxFireRange);

    // 准备射线检测参数
    FHitResult HitResult;
    FCollisionQueryParams QueryParams;
    QueryParams.AddIgnoredActor(this);  // 忽略自己（防止打到自己）
    QueryParams.bTraceComplex = true;   // 使用复杂碰撞（更精确）

    // 执行射线检测
    bool bHit = GetWorld()->LineTraceSingleByChannel(
        HitResult,
        EyeLocation,
        EndLocation,
        ECC_Visibility,    // 使用可见性碰撞通道
        QueryParams
    );

    // 4. 处理命中结果
    if (bHit)
    {
        AActor* HitActor = HitResult.GetActor();
        if (HitActor)
        {
            // 判断是否命中了另一个玩家角色
            ANetworkCharacter* HitCharacter = Cast<ANetworkCharacter>(HitActor);
            if (HitCharacter && HitCharacter->bIsAlive)
            {
                // ===== 命中了其他玩家！ =====

                // 判断是否是爆头（命中了头部骨骼或头部碰撞体）
                // 简化判断：通过命中位置的 Z 高度
                bool bIsHeadshot = HitResult.BoneName.ToString().Contains(TEXT("head"))
                    || HitResult.BoneName.ToString().Contains(TEXT("Head"));

                // 计算实际伤害
                float ActualDamage = WeaponDamage;
                if (bIsHeadshot)
                {
                    ActualDamage *= HeadshotMultiplier;  // 爆头两倍伤害
                    UE_LOG(LogTemp, Log, TEXT("服务器：爆头！伤害 %.0f"), ActualDamage);
                }

                // 对被击中的角色造成伤害（在服务器上执行）
                HitCharacter->ServerApplyDamage(ActualDamage, this, bIsHeadshot);
            }

            // 播放命中特效（所有客户端看到火花）
            MulticastPlayHitEffect(HitResult.Location, false);
        }
    }

    // 5. 可选：生成子弹投射物（如果使用投射物系统而非即时命中）
    // 这里用的是即时命中（Hitscan），如果需要投射物：
    // if (ProjectileClass)
    // {
    //     // 在服务器上生成子弹
    //     ANetworkProjectile* Projectile = GetWorld()->SpawnActor<ANetworkProjectile>(
    //         ProjectileClass,
    //         EyeLocation,
    //         AimDirection.Rotation()
    //     );
    //     // 子弹会自动飞行并在服务器上检测碰撞
    // }
}

// ============================================
// 服务器端伤害处理
// ============================================
void ANetworkCharacter::ServerApplyDamage(
    float Damage,
    ANetworkCharacter* DamageInstigator,
    bool bIsHeadshot
)
{
    // ===== 第一行：检查权限 =====
    if (!HasAuthority())
    {
        UE_LOG(LogTemp, Error, TEXT("ServerApplyDamage 在客户端被调用！这是不允许的！"));
        return;
    }

    // ===== 检查：目标是否还活着 =====
    if (!bIsAlive)
    {
        return;  // 已经死了
    }

    // ===== 护甲伤害吸收 =====
    // 护甲可以吸收一部分伤害
    float DamageToHealth = Damage;
    if (Armor > 0.0f)
    {
        // 护甲吸收 40% 的伤害
        float ArmorAbsorption = FMath::Min(Armor, Damage * 0.4f);
        Armor -= ArmorAbsorption;          // 护甲减少
        DamageToHealth -= ArmorAbsorption;  // 剩余伤害作用于血量
    }

    // ===== 应用伤害到血量 =====
    // FMath::Max(0, ...) 确保血量不会低于 0
    Health = FMath::Max(0.0f, Health - DamageToHealth);

    UE_LOG(LogTemp, Log, TEXT("服务器：%s 受到 %.0f 伤害（来自 %s），剩余血量 %.0f，护甲 %.0f"),
        *GetName(),
        DamageToHealth,
        DamageInstigator ? *DamageInstigator->GetName() : TEXT("Unknown"),
        Health,
        Armor
    );

    // ===== 通知客户端 =====
    // Client RPC：通知被击中的玩家显示伤害数字
    ClientShowDamageNumber(DamageToHealth, bIsHeadshot);

    // ===== 死亡检测 =====
    if (Health <= 0.0f)
    {
        ServerHandleDeath(DamageInstigator);
    }
}

// ============================================
// 服务器端死亡处理
// ============================================
void ANetworkCharacter::ServerHandleDeath(ANetworkCharacter* Killer)
{
    if (!HasAuthority())
    {
        return;
    }

    // ===== 设置死亡状态 =====
    bIsAlive = false;
    DeathCount++;

    // ===== 通知杀死者增加击杀数 =====
    if (Killer && Killer != this)  // 确保不是自杀
    {
        Killer->KillCount++;

        // 通知击杀者（Client RPC）
        Killer->ClientShowKillConfirmation(GetName());

        UE_LOG(LogTemp, Log, TEXT("服务器：%s 击杀了 %s（击杀数：%d）"),
            *Killer->GetName(),
            *GetName(),
            Killer->KillCount
        );
    }
    else
    {
        // 自杀或环境击杀
        UE_LOG(LogTemp, Log, TEXT("服务器：%s 死亡（自杀/环境）"), *GetName());
    }

    // ===== 通知所有客户端播放死亡动画 =====
    MulticastPlayDeathAnimation();

    // ===== 禁用碰撞和移动 =====
    // 死亡的角色不应该有碰撞（可以被穿过）
    GetCapsuleComponent()->SetCollisionEnabled(ECollisionEnabled::NoCollision);

    // 禁用输入
    if (APlayerController* PC = Cast<APlayerController>(GetController()))
    {
        DisableInput(PC);
    }

    // ===== 设置重生计时器 =====
    // 5秒后自动重生（实际项目中通常在 GameMode 中处理）
    FTimerHandle RespawnTimer;
    GetWorldTimerManager().SetTimer(
        RespawnTimer,
        this,
        &ANetworkCharacter::ServerRespawn,
        5.0f,    // 5秒后重生
        false
    );
}

// ============================================
// 服务器端重生处理
// ============================================
void ANetworkCharacter::ServerRespawn()
{
    if (!HasAuthority())
    {
        return;
    }

    // ===== 恢复状态 =====
    Health = MaxHealth;
    Armor = MaxArmor;
    Ammo = MaxAmmo;
    bIsAlive = true;

    // ===== 恢复碰撞 =====
    GetCapsuleComponent()->SetCollisionEnabled(ECollisionEnabled::QueryAndPhysics);

    // ===== 恢复输入 =====
    if (APlayerController* PC = Cast<APlayerController>(GetController()))
    {
        EnableInput(PC);
    }

    // ===== 传送到出生点 =====
    // 查找 PlayerStart 并传送
    TArray<AActor*> PlayerStarts;
    UGameplayStatics::GetAllActorsOfClass(
        GetWorld(),
        APlayerStart::StaticClass(),
        PlayerStarts
    );
    if (PlayerStarts.Num() > 0)
    {
        // 随机选一个出生点
        int32 RandomIndex = FMath::RandRange(0, PlayerStarts.Num() - 1);
        SetActorLocation(PlayerStarts[RandomIndex]->GetActorLocation());
        SetActorRotation(PlayerStarts[RandomIndex]->GetActorRotation());
    }

    // ===== 通知所有客户端播放重生效果 =====
    MulticastPlayRespawnEffect();

    UE_LOG(LogTemp, Log, TEXT("服务器：%s 已重生"), *GetName());
}

// ============================================
```
```cpp
// NetworkProjectile.cpp
// 服务器权威的子弹实现
// ============================================
#include "NetworkProjectile.h"
#include "NetworkCharacter.h"
#include "Net/UnrealNetwork.h"
#include "Components/SphereComponent.h"
#include "Components/StaticMeshComponent.h"
#include "GameFramework/ProjectileMovementComponent.h"
#include "Engine/Engine.h"

// ============================================
// 构造函数
// ============================================
ANetworkProjectile::ANetworkProjectile()
{
    // ===== 网络设置 =====
    bReplicates = true;                    // ★ 子弹必须通过网络同步
    NetUpdateFrequency = 60.0f;            // 子弹高速移动，需要高频率更新
    NetPriority = 3.0f;                    // 高优先级

    // ★★★ 重要：设置复制移动 ★★★
    // 这使得引擎自动同步子弹的位置和旋转
    SetReplicateMovement(true);

    // ===== 创建碰撞球体 =====
    CollisionComponent = CreateDefaultSubobject<USphereComponent>(TEXT("CollisionSphere"));
    CollisionComponent->InitSphereRadius(5.0f);  // 半径为5cm

    // 碰撞设置：
    // QueryOnly：只检测碰撞（不产生物理弹开效果）
    CollisionComponent->SetCollisionEnabled(ECollisionEnabled::QueryOnly);
    CollisionComponent->SetCollisionObjectType(ECC_WorldDynamic);
    CollisionComponent->SetCollisionResponseToAllChannels(ECR_Block);  // 默认阻挡所有
    // ↓ 忽略开火者（防止打到自己）
    // CollisionComponent->SetCollisionResponseToChannel(ECC_Pawn, ECR_Ignore);
    RootComponent = CollisionComponent;

    // ===== 创建子弹模型 =====
    MeshComponent = CreateDefaultSubobject<UStaticMeshComponent>(TEXT("ProjectileMesh"));
    MeshComponent->SetupAttachment(RootComponent);
    MeshComponent->SetCollisionEnabled(ECollisionEnabled::NoCollision);  // 模型不参与碰撞

    // ===== 创建子弹移动组件 =====
    ProjectileMovement = CreateDefaultSubobject<UProjectileMovementComponent>(TEXT("ProjectileMovement"));
    ProjectileMovement->UpdatedComponent = CollisionComponent;
    ProjectileMovement->InitialSpeed = 3000.0f;  // 初始速度 30米/秒
    ProjectileMovement->MaxSpeed = 3000.0f;
    ProjectileMovement->bRotationFollowsVelocity = true;  // 子弹跟随速度方向旋转
    ProjectileMovement->bShouldBounce = false;             // 子弹不反弹
    ProjectileMovement->ProjectileGravityScale = 0.0f;     // 无重力（直线飞行）

    // ===== 默认配置 =====
    Damage = 25.0f;
    Speed = 3000.0f;
    LifeSpan = 10.0f;  // 10秒后自动销毁

    // ===== 设置 Actor 的生命周期 =====
    // 子弹在飞行 LifeSpan 秒后自动销毁（无论是否命中）
    InitialLifeSpan = LifeSpan;
}

// ============================================
// BeginPlay
// ============================================
void ANetworkProjectile::BeginPlay()
{
    Super::BeginPlay();

    // ===== 只在服务器上绑定碰撞事件 =====
    // 原因：只有服务器才需要检测碰撞并计算伤害
    // 客户端不需要绑定——客户端只看服务器同步的位置
    if (HasAuthority())
    {
        // 绑定碰撞回调函数
        CollisionComponent->OnComponentHit.AddDynamic(this, &ANetworkProjectile::OnProjectileHit);

        // 记录拥有者（用于后续伤害处理时不伤害自己）
        // OwnerCharacter = Cast<ANetworkCharacter>(GetOwner());

        UE_LOG(LogTemp, Log, TEXT("服务器：子弹已生成，速度 %.0f"), Speed);
    }
    else
    {
        // 客户端只需要看到子弹飞行
        UE_LOG(LogTemp, Verbose, TEXT("客户端：看到一颗子弹"));
    }
}

// ============================================
// 复制属性注册
// ============================================
void ANetworkProjectile::GetLifetimeReplicatedProps(
    TArray<FLifetimeProperty>& OutLifetimeProps
) const
{
    Super::GetLifetimeReplicatedProps(OutLifetimeProps);

    // 同步飞行方向和初始速度（用于客户端的外推/插值）
    DOREPLIFETIME_CONDITION(ANetworkProjectile, FlightDirection, COND_InitialOnly);
    DOREPLIFETIME_CONDITION(ANetworkProjectile, InitialSpeed, COND_InitialOnly);
    // ↑ COND_InitialOnly：只在子弹第一次复制时同步一次
    // 因为方向和速度不会改变（直线飞行）
}

// ============================================
// 碰撞回调 — 子弹命中了什么
// ============================================
void ANetworkProjectile::OnProjectileHit(
    UPrimitiveComponent* HitComponent,
    AActor* OtherActor,
    UPrimitiveComponent* OtherComp,
    FVector NormalImpulse,
    const FHitResult& Hit
)
{
    // ===== 第一步：权限检查 =====
    // ★★★ 这个判断最重要！★★★
    // 碰撞检测只应该在服务器上触发实际逻辑
    // 客户端上的碰撞回调不应该产生任何游戏影响
    if (!HasAuthority())
    {
        // 客户端上的碰撞事件——忽略！
        // 客户端可能因为预测/插值的位置偏差产生碰撞
        // 但客户端不能决定"这个子弹是否命中了"
        UE_LOG(LogTemp, Verbose, TEXT("客户端：忽略碰撞事件（无权威）"));
        return;
    }

    // ===== 第二步：服务器端命中处理 =====
    // 从这一行往下，代码只在服务器上执行
    HandleHitOnServer(Hit);
}

// ============================================
// 服务器端命中处理
// ============================================
void ANetworkProjectile::HandleHitOnServer(const FHitResult& Hit)
{
    // ===== 确保在服务器上 =====
    // （这个检查虽然在这个例子中多余，但在大型项目中是好习惯）
    if (!HasAuthority())
    {
        return;
    }

    AActor* HitActor = Hit.GetActor();
    if (!HitActor)
    {
        return;  // 没有命中任何 Actor
    }

    UE_LOG(LogTemp, Log, TEXT("服务器：子弹命中 %s"), *HitActor->GetName());

    // ===== 检查是否命中了玩家角色 =====
    ANetworkCharacter* HitCharacter = Cast<ANetworkCharacter>(HitActor);
    if (HitCharacter)
    {
        // ===== 验证：不伤害自己 =====
        // if (HitCharacter == OwnerCharacter)
        // {
        //     UE_LOG(LogTemp, Log, TEXT("服务器：子弹忽略自己"));
        //     return;  // 不能打自己
        // }

        // ===== 验证：目标是否存活 =====
        if (!HitCharacter->bIsAlive)
        {
            UE_LOG(LogTemp, Log, TEXT("服务器：目标已死亡，子弹穿过"));
            return;  // 死人不会受伤害
        }

        // ===== 造成伤害 =====
        // 调用目标的 ServerApplyDamage（只在服务器执行）
        ANetworkCharacter* InstigatorChar = Cast<ANetworkCharacter>(GetOwner());
        HitCharacter->ServerApplyDamage(Damage, InstigatorChar, false);

        UE_LOG(LogTemp, Log, TEXT("服务器：子弹对 %s 造成了 %.0f 伤害"),
            *HitCharacter->GetName(), Damage);
    }

    // ===== 子弹命中后销毁 =====
    // 在服务器上销毁子弹（客户端会同步销毁）
    Destroy();
}
```

```cpp
    FPlayerStats* Stats = FindPlayerStats(PlayerName);
    if (Stats)
    {
        Stats->Kills = Kills;
        Stats->Score = Kills * 100 - Stats->Deaths * 10;  // 简单计分公式
    }
}

{
    FPlayerStats* Stats = FindPlayerStats(PlayerName);
    if (Stats)
    {
        Stats->Deaths = Deaths;
        Stats->Score = Stats->Kills * 100 - Deaths * 10;
    }
}

FPlayerStats* ANetworkGameState::FindPlayerStats(const FString& PlayerName)
{
    for (FPlayerStats& Stats : PlayerStatsArray)
    {
        if (Stats.PlayerName == PlayerName)
        {
            return &Stats;
        }
    }
    return nullptr;
}
```

### 常见问题排查

| 问题 | 可能原因 | 解决方法 |
|------|---------|---------|
| 客户端看不到其他玩家 | 1. 属性没有 Replicated<br>2. Actor 没有 bReplicates = true<br>3. 没有注册 GetLifetimeReplicatedProps | 检查这三项是否正确配置 |
| RPC 不执行 | 1. 没有在头文件中声明 UFUNCTION(Server/Client/NetMulticast)<br>2. 函数名不符合命名约定<br>3. 忘记实现 _Implementation 函数 | 检查声明和实现 |
| 客户端上能修改血量 | HasAuthority() 检查缺失 | 在修改血量的代码前加上 if (!HasAuthority()) return; |
| 多人不同步/位置瞬移 | NetUpdateFrequency 太低 | 提高 NetUpdateFrequency（如30-60） |
| 编译错误：未定义类型 | 忘记包含 Net/UnrealNetwork.h | 在 .cpp 中加入 #include "Net/UnrealNetwork.h" |
