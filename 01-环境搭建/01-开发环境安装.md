# 1.1 开发环境安装

> **目标**：在你的Windows电脑上装好UE5和Visual Studio，成功创建一个C++项目并运行。

---

## 为什么这些软件是必须的？

| 软件                    | 干什么用的                                 | 能不能不用？                                        |
| ----------------------- | ------------------------------------------ | --------------------------------------------------- |
| **Unreal Engine 5**     | 游戏引擎本体，提供编辑器、渲染、物理等一切 | ❌ 不能，这是UE5 C++开发，不是纯C++开发             |
| **Visual Studio 2022**  | 写C++代码的IDE，并安装MSVC工具链           | ❌ 不能只装VS Code，UE需要MSVC/Windows SDK工具链    |
| **Epic Games Launcher** | 下载UE5、管理引擎版本                      | ✅ 可以从官网直接下载引擎源码编译，但不推荐新手尝试 |

---

## 第一步：注册Epic Games账号

1. 打开浏览器，访问 **https://www.unrealengine.com**
2. 点击右上角 **"Download"** 或 **"Get Started"**
3. 选择 **"Creative / Gaming"** 类别的账号（个人使用免费）
4. 注册完成后登录

> **注意**：学习阶段使用UE5通常不需要付费。商业发布时，游戏类产品通常是在单个产品生命周期总收入超过100万美元后，对超过门槛的部分按协议计算版税；非游戏商业用途还可能涉及席位订阅。授权规则会更新，正式发布前务必以Epic官网的当前许可协议为准。

---

## 第二步：安装Epic Games Launcher

1. 在UE官网点击 **Download** 按钮
2. 下载 **Epic Games Launcher** 安装程序（约50MB）
3. 双击运行安装程序
4. 一路默认选项安装，安装路径用默认的即可
5. 安装完成后启动，用刚才注册的账号登录

> **遇到问题？** 如果启动器闪退或报错，尝试右键→以管理员身份运行。

---

## 第三步：安装Unreal Engine 5

1. 在Epic Games Launcher中，点击左侧 **"Unreal Engine"** 标签
2. 点击顶部 **"Library"** （库）标签
3. 点击 **"+"** 号旁边的引擎版本按钮
4. 在下拉菜单中选择一个**非Preview的稳定版**（例如当前Launcher中显示的最新正式版）

   > **版本说明**：数字越大通常越新，但不同小版本的菜单、默认模板和工具链要求可能有差异。本手册面向UE5稳定版；如果界面文字略有不同，以当前编辑器中最接近的菜单项为准。不要选带Preview的预览版。

5. 点击 **"Install"** 按钮
6. 在弹出的选项中：

   | 选项                             | 建议                                                  |
   | -------------------------------- | ----------------------------------------------------- |
   | **安装路径**                     | 确保至少有 **60GB** 空闲空间。建议安装到SSD固态硬盘   |
   | **Core Components**              | 默认全选，不要取消                                    |
   | **Starter Content**              | ✅ 勾选（新手学习用）                                 |
   | **Templates and Feature Packs**  | ✅ 勾选（包含项目模板）                               |
   | **Target Platforms**             | 至少勾选 `Win64`。如果你要做Android/iOS再勾选对应平台 |
   | **Editor symbols for debugging** | ✅ 建议勾选（调试C++代码需要，但会多占约50GB）        |

7. 点击 **"Install"** 开始下载

> **下载时间**：根据网速不同，可能需要30分钟到数小时。引擎本体约25GB，加上调试符号和平台支持可能到50GB+。建议睡前开始下载。

### 安装完成后验证

1. 在Epic Games Launcher → Unreal Engine → Library中，你应该能看到引擎版本号旁边显示绿色的 **"Launch"** 按钮
2. 点击 **"Launch"** 启动UE5编辑器
3. 如果成功看到项目选择界面，说明引擎安装成功
4. 先关闭编辑器，我们还需要装Visual Studio

---

## 第四步：安装Visual Studio 2022

本手册的Windows环境统一使用 **Visual Studio 2022 + MSVC v143工具链**。Rider等IDE也可以写UE C++，但仍然需要安装UE版本要求的MSVC和Windows SDK。

### 4.1 下载Visual Studio 2022

1. 打开浏览器，访问 **https://visualstudio.microsoft.com/zh-hans/downloads/**
2. 找到 **Visual Studio 2022** 部分
3. 点击 **"Community 2022"** 下面的 **"免费下载"** 按钮
   - Community版完全免费，功能足够专业开发使用
   - Professional和Enterprise版需要付费但功能相同（多了团队管理功能）

### 4.2 安装时的工作负载选择（重要！）

1. 双击运行安装程序
2. 在 **"工作负载"** 标签页中，建议勾选以下项目：
   - ✅ **"使用C++的游戏开发"**（Game development with C++）
     - 这会在右侧显示额外组件，确保 **Unreal Engine安装程序** 也被勾选
   - ✅ **"使用C++的桌面开发"**（Desktop development with C++）
   - ✅ **".NET桌面开发"**（部分UE工具链/项目生成流程会用到.NET运行时）

   > **说明**：不要为了“保险”把所有工作负载都勾上。对本教程来说，上面这些已经足够；移动端、MAUI等工作负载只有在你明确要开发对应平台时再安装。

3. 在右侧 **"单个组件"** 标签页中，确保以下被勾选：
   - ✅ **MSVC v143 - VS 2022 C++ x64/x86 生成工具（最新）**

   以下为可选项，根据你的引擎版本安装
   - **MSVC v143 - VS 2022 C++ x64/x86 生成工具（v14.44-17.14）**
   - **MSVC v143 - VS 2022 C++ x64/x86 生成工具（v14.36-17.6）**
   - **MSVC v143 - VS 2022 C++ x64/x86 生成工具（v14.38-17.8）**
   - **MSVC v143 - VS 2022 C++ x64/x86 生成工具（v14.34-17.4）**
   - **MSVC v143 - VS 2022 C++ x64/x86 生成工具（v14.31-17.1）**

   UE5 各版本对应的 MSVC v143（VS 2022）生成工具版本

   根据 Cesium for Unreal 文档的说明，UE 的首选 MSVC 版本记录在引擎安装目录的 `Engine/Config/Windows/Windows_SDK.json` 中的 `PreferredVisualCppVersions` 字段，各版本对应关系如下：

   | UE 版本   | 首选/推荐 MSVC 版本                     | 对应 VS 2022 版本 | MSVC v143 组件标签 |
   | --------- | --------------------------------------- | ----------------- | ------------------ |
   | **5.0.x** | 14.31.x（VS2019 为主，VS2022 初步支持） | VS 2022 17.1+     | v14.31-17.1        |
   | **5.1.x** | 14.34.x                                 | VS 2022 17.4      | v14.34-17.4        |
   | **5.2.x** | 14.36.32532                             | VS 2022 17.6      | v14.36-17.6        |
   | **5.3.x** | 14.34.31933（最低要求），推荐 14.36.x   | VS 2022 17.4~17.6 | v14.36-17.6        |
   | **5.4.x** | **14.38.33130**                         | VS 2022 17.8      | v14.38-17.8        |
   | **5.5.x** | **14.38.33130**                         | VS 2022 17.8      | v14.38-17.8        |
   | **5.6.x** | **14.38.33130**                         | VS 2022 17.8      | v14.38-17.8        |
   | **5.7.x** | **14.44.35207**                         | VS 2022 17.14     | v14.44-17.14       |

4. **安装位置**：建议装到C盘避免出现兼容性问题，如果C盘空间不足的话可以把IDE装C盘、SDK和工具链装D盘
5. 点击 **"安装"** 按钮
6. 等待安装完成（约20-40分钟，取决于网速和磁盘速度）

### 4.3 验证VS2022安装

1. 从开始菜单启动 **Visual Studio 2022**
2. 首次启动会要求登录，用你的Microsoft账号（没有就注册一个）或者跳过
3. 选择开发设置：选 **"C++"** 或 **"游戏开发"**
4. 看到VS主界面就表示安装成功
5. 关闭VS（不需要手动创建项目，UE5会帮我们创建）

---

## 第五步：关联UE5和Visual Studio

这一步确保UE5知道用VS2022来编译C++代码。

1. 打开Epic Games Launcher
2. 启动UE5（点击对应版本的 Launch 按钮）
3. 在UE5项目浏览器中，选择一个 **C++项目模板**（例如Games→Third Person）来测试
4. 点击 **Create** 创建项目
5. UE5会自动调用VS2022来编译初始项目代码
6. **如果编译成功**：UE编辑器会自动打开你的项目 ✅
7. **如果编译失败**：看下面的常见问题排查

## 常见问题排查

### Q1: UE5创建C++项目时报错 "Could not find Visual Studio"

**原因**：UE没找到VS2022。

**解决**：

1. 确保安装了"使用C++的游戏开发"工作负载
2. 打开VS2022 → 工具 → 获取工具和功能 → 确认工作负载已勾选
3. 重启Epic Games Launcher和UE
4. 如果还不行，到UE安装目录找 `Engine\Extras\Redist\en-us\` 运行 `UEPrereqSetup_x64.exe`

### Q2: 编译时提示 "error MSB8036: The Windows SDK version xxx was not found"

**原因**：Windows SDK版本不匹配。

**解决**：

1. 打开VS2022安装程序 → 修改 → 单个组件
2. 搜索 "Windows SDK"，安装最新版本
3. 或者：在项目 `.Target.cs` 文件中修改Windows SDK版本号

### Q3: 编译特别慢，C盘满了

**原因**：UE编译会产生大量中间文件。

**解决**：

1. 项目目录下的 `Intermediate`、`Saved`、`DerivedDataCache` 可以删除（下次编译会重新生成）
2. 考虑把项目放在非系统盘
3. 引擎安装到SSD、项目放SSD

### Q4: 启动UE编辑器时黑屏或闪退

**原因**：通常是显卡驱动问题。

**解决**：

1. 更新显卡驱动（NVIDIA/AMD官网下载最新驱动）
2. 右键桌面 → 显示设置 → 图形设置 → 把UE5Editor.exe设为"高性能"
3. 如果还不行，启动UE5时加命令行参数 `-d3d11`（使用DX11而非DX12）
